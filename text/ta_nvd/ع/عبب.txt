( وَعبُّ الشَّمْسِ ) بالتَّشْدِيدِ عَلى قَوْله بَعْض ( ويُخَفَّفُ ) وهُو المَعْرُوفُ المَشْهُورُ ( ضَوْؤُهَا ) أَي الشَّمْس ، وقَالَ الأَزْهَرِيُّ : عَبُّ الشَّمْسِ : ضَوْءُ الصُّبْحِ وَعَلَى التَّخْفِيفِ قَالَ الشَّاعِر :<br>\
ورَأْسُ عَبِ الشَّمْسِ المَخُوفُ ذِمَاؤُهَا<br>\
وقَال الأَزْهَرِيّ في عَبْقَر عِنْدَ إِنْشَادِه :<br>\
كَأَنَّ فَاهَا عَبُّ قُرَ بَارِدِ<br>\
قال : وبِهِ سُمِّي عَبْشَمْسٌ .<br>\
وفِي لِسَانِ الْعَرَبِ : وَقَوْلُهُم : عَبُّ شَمْس أَرَادُا عَبْدَ شَمْسٍ .<br>\
قال ابن شُمَيْل : وفي سَعْدٍ بَنُو عَبِّ الشَّمْسِ ، وفي قُرَيْشٍ بَنُو عَبْدِ الشمْسِ .<br>\
( وذُو عُبَبٍ كصُرَد : وَادٍ ) .<br>\
( والعُبَبُ : حَبُّ الكَاكَنْج ) ، وإِنَّمَا لم يَضْبطْه اعْتِمَاداً عَلَى ضَبْطِ مَا قَبْلَه ، وأَخْطَأَ مَنْ رَأَى ظَاهِرَ الإِطْلَاق فَضَبَطَه مُحَرَّكَةً ، ثم إِن الكَاكَنْج ، عَلى مَا قَالَه غَيرُ وَاحِدٍ من الأَئِمَّة : شَجَرٌ ، والعَبَبُ حَبُّه ، ويَأْتِي في كَلَامِ المُؤَلِّف أَنَّه صَمْغٌ ، فَتَأَمَّل .<br>\
أَشَارَ لِذَلِكَ شَيْخُنَا ، ( أَو عِنَبُ<br>\
الثَّعْلَبِ ) قَالَهُ ابْنُ الأَعْرَابِيّ .<br>\
قال ابنُ حَبِيب : هُوَ العُبَبُ ومَنْ قَالَ : عِنَبُ الثَّعْلَب فَقَدْ أَخْطَأَ .<br>\
قالَ أَبُو مَنْصُور : عنَبُ الثَّعْلَبِ صَحِيحٌ ولَيْسَ بِخَطَإٍ .<br>\
ووجَدْتُ بَيْتاً لأَبِي وَجزْةً يَدُلُّ عَلَى مَا قَالَه ابنُ الأَعْرَابِيّ :<br>\
إِذَا تَرَبَّعْتَ مَا بَيْنَ الشُّرَيْقِ إِلَى<br>\
رَوْضِ الفِفَاج أُولَات السَّرْح والعُبَبِ<br>\
( أَو ) شَجَرَةٌ يُقَالُ لَهَا ( الرَّاءُ ) مَمْدُوداً ، قَالَهُ ابْنُ لأَعْرَابِيّ ، ( أَو ) ضَرْبٌ من النَّبَات ، وَزَعَمَ أَبُو حَنِيفة أَنِ ( شَجَرَةٌ مِنَ الأَغْلَاثِ ) تخُبِهُ الحَرْمَل إِلَّا أَنَّهَا أَطولُ في السَّماءِ تَخْرُج خَيطَاناً ولها سِنَفَةٌ مِثْلُ سِنَفَةِ الحَرْمَلِ وقد تَقْضَمُ المِعْزَى مِنْ وَرَقِهَ ومن سِنَفَتِهَا إِذَا يَبِسَت .<br>\
( و ) العُبُبُ ( بِضَمَّتَيْن : المِيَاهُ المُنْدَفِقَةُ ) وفي نُسْخَة المُتَدَفِّقَةُ ، قَاله ابنُ الأَعْرَابِيّ .<br>\
( وعَبْعَبَ ) إِذَا ( انْهَزَم ) .<br>\
وَعَبَّ إِذَا حَسُنَ وَجْهُه بَعْدَ تَغَيُّر .<br>\

( و ) في النَّوَادِر يُقَال : ( تَعَبْعَبْتُه ) أَي الشَّيءَ وتَوَعَّبْتُه واسْتَوْعَبْتُه وتَقَمْقَمْتُه وتَضَمَّمْتُه ( أَي أَتَيْتُ عَلَيْهِ كُلِّه ) .<br>\
