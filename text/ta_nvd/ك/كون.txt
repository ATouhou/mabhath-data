( و ) مِن أَقْسامِ كانَ الناقِصَة أَنْ تأْتي ( بمَعْنَى صارَ : كقَوْلِه تعالَى : { وكانَ مِن الكَافِرينَ } .<br>\
قالَ ابنُ بَرِّي : ومنه قَوْلُه تعالى أَيْضاً : { كُنْتُم خَيْرَ أُمَّةٍ } ومنه قَوْلُه تعالى : { فإذا انْشَقَّتِ السَّماءُ فكانتْ وَرْدَةً كالدِّهانِ } ؛ وقَوْلُه تعالى : { وكانتُ الجِبالُ كَثِيباً مَهِيلاً } ؛ وقَوْلُه تعالى : { وما جَعَلْنا القِبْلَةَ التي كُنْتَ عليها } ، أَي صِرْتَ إليها : وقَوْلُه تعالى : { كيفَ نُكَلِّمُ مَن كانَ في المَهْدِ صَبِيّاً } ؛ وقالَ شَمْعَلَةُ بنُ الأَخْضَر :<br>\
فَخَرَّ على الأَلاءَةِ لم يُوَسَّدْوقد كانَ الدِّماءُ له خِمارَا قلْتُ : ومنه أَيْضاً في حدِيثِ كَعْبٍ ، رضِيَ اللّهُ تعالى عنه : ( كُنْ أَبا خَيْثَمَة ) ، أَي صِرْهُ ؛ يقالُ للرَّجُلِ يُرَى مِن بُعْد : كُنْ فُلاناً ، أَي أَنْتَ فلانٌ ، أَو هو فُلانٌ .<br>\
وقالَ أَبو العبَّاسِ : اخْتَلَفَ الناسُ في قوْلِه تعالىَ : { كيفَ نُكَلِّمُ مَنْ كانَ في المَهْدِ صَبِيّاً } ، فقالَ بعضُهم : كانَ هنا صِلَة ، ومَعْناه كيفَ نُكَلِّمُ مَنْ هو في المَهْدِ صَبِيّاً ؛ وقالَ الفرَّاءُ : كانَ هنا شَرْطٌ وفي الكَلامِ تَعجُّبٌ ، ومَعْناه : مَنْ يَكُنْ في المَهْدِ صَبِيّاً فكيفَ يُكَلَّمُ ؟ .<br>\
( وبمَعْنَى ( الاسْتِقْبالِ : كقَوْلِه تعالَى : { يَخافونَ يوماً كانَ شَرُّهُ<br>\
مُسْتَطِيراً } ، ومنه قَوْلُ الطرمَّاحِ :<br>\
وإنّي لآتِيكُم تَشَكُّرَ ما مَضَى من الأَمْرِ واسْتِنْجاز ما كانَ في غَدِو قَوْلُ سَلَمَة الجُعْفِيّ :<br>\
وكُنْتُ أَرَى كالمَوْتِ من بَيْنِ سَاعَةٍ فكيفَ بِبَيْنٍ كانَ مِيعادُه الحَشْرَا ؟ ( وبمَعْنَى المُضِيِّ المُنْقَطِعِ وهي التَّامَّةُ كقَوْلِه تعالَى : { وكانَ في المَدينَةِ تِسْعَةُ رَهْطٍ } يُفْسِدُونَ ، ومنه قَوْلُ أَبي الغول :<br>\
عَسَى الأيامُ أَن يَرْجِعْنَ قَوماً كالذي كانوا أَي مَضَوْا وانْقَضَوا : وقَوْلُ أَبي زُبَيْدٍ :<br>\
ثم أَضْحَوْا كأنَّهُم لم يَكُونوا ومُلُوكاً كانوا وأَهْلَ عَلاءِ ( وبمَعْنَى الحالِ : كقَوْلِهِ تعالى : { كُنْتُمْ خيرَ أُمَّةٍ } أُخْرِجَتْ للنَّاسِ ؛ ورُوِي عن ابنِ الأَعْرابيِّ في تَفْسيرِ هذه الآيَة قالَ : أَي أَنْتُم خَيْر أُمَّةٍ ، قالَ : ويقالُ مَعْناه كُنْتُم خَيْرَ أُمَّةٍ في عِلْمِ اللّهِ .<br>\
وعليه خَرَّجَ بعضٌ قَوْلَه تعالى : { وكانَ اللّهُ غَفْوراً حِيماً } ، لأنَّ كانَ بمَنْزلَةِ ما في الحالِ ، والمعْنَى : واللّهُ غَفُورٌ رَحِيمٌ إلاَّ أَنْ كَوْنَ الماضِي بمعْنَى الحالِ قَلِيلٌ .<br>\
واحْتَجّ صاحِبُ هذا القَوْلِ بقَوْلِهم : غَفَرَ اللّهُ لفلانٍ ، بمعْنَى لِيَغْفِر اللّهُ ، فلمَّا كانَ في الحالِ دَليلٌ على الاسْتِقْبالِ وَقَعَ الماضِي مُؤَدِّياً عنها اسْتِخْفافاً لأنَّ اخْتِلافَ أَلْفاظِ الأفْعالِ إنَّما وَقَعَ لاخْتِلافِ الأَوْقاتِ ؛ ومنه قَوْلُ أَبي جُنْدبٍ الهُذَليِّ :<br>\
وكنتُ إذا جارِي دعا لمَضُوفةٍ أُشَمِّرُ حتى يَنْصُفَ الساقَ مِئْزَرِي وإنَّما يخبر عن حالِهِ لا عمَّا مَضَى مِن فِعْلِه .<br>\
( وكَيْوانُ زُحَلُ مَمْنُوعٌ مِن الصَّرْفِ ، والقَوْلُ فيه كالقَوْلِ في خَيْوانَ ، والمانِعُ له مِن الصَّرْفِ العُجْمة ، كما أنَّ المانِعَ لخَيْوان مِن الصَّرفِ إنَّما هو التّأْنِيثُ وإرادَةُ البُقْعَة أَو الأَرْض أَو القَرْجيَةِ ، وسَيَأْتي .<br>\
( وسَمْعُ الكِيانِ : كتابٌ للعَجَمِ .<br>\

( والاسْتِكانَةُ : الخُضوعُ والذّلُّ .<br>\
جَعَلَهُ بعضُهم اسْتَفْعَلَ من الكَوْنِ ، وجَعَلَهُ أَبو عليَ مِنَ الكَيْنِ وهو الأشْبَهُ .<br>\

( والمَكانَةُ : المَنْزِلَةُ ؛ نَقَلَهُ الجَوْهرِيُّ .<br>\
وتقدَّمَ كَلامُ ابنُ بَرِّي قَرِيباً في الرَّدِّ عليه .<br>\

( والتَّكَوُّنُ : التَّحَرُّكُ ؛ عن ابنِ الأعْرابيِّ ، قالَ : ( وتَقُولُ العَرَبُ ( للبَغِيضِ : لا كانَ ولا تَكَوَّنَ ، أَي لا خُلِقَ ولا تَحَرَّكَ ، أَي ماتَ .<br>\
