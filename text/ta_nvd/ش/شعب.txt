والفِئَامُ في الْمَزَايِد : أَن يُؤْخَذَ الأَدِيمُ فيُثنَى .<br>\
ثم يُزَادَ في جَوَانِبِها ما يُوَسِّعُها .<br>\
قال الرَّاعِي يَصِفُ إِبِلاً تَرْعَى في العَزِيبِ :<br>\
إِذا لَمْ تَرُحْ أعدَّى إِلَيْهَا مُعَجِّلٌ<br>\
شَعِيبَ أَدِيم ذَا فِرَاغَيْنِ مْتْرَعا<br>\
يعني ذا أَدِيمَيْن قُوِبلَ بينهما وقيل : التي تُفْأَمْ بِجِلْدٍ ثالِثٍ بَيْنَ الجِلْدَيْن لتَتَّسِعَ .<br>\
وقِيلَ : هِي التي من قِطْعَتَيْن شُعِبَتْ إحْدَاهُمَا إِلَى الأُخْرَى أَي ضُمَّت .<br>\
( أَو ) هِيَ ( المَخْرُوزَةُ مِنْ وَجْهَيْن ) وكُلُّ ذلِكَ من الجَمْع .<br>\
( و ) الشَّعِيبُ أَيضاً : ( السِّقَاءُ البَالِي ) لأَنَّه يُشْعَبُ .<br>\
( ج ) أَي جَمْعُ كُلّ ذَلِك شُعُب ( كَكُتُبٍ ) .<br>\
وفي لِسَانِ العَرَب : الشَّعِيبُ والمَزَادَةُ والرَّاوِيَة السَّطِيحَةُ شَيْء واحد ، سُمِّي بِذلكِ لأَنَّه ضُمَّ بَعْضُه إِلَى بَعْض .<br>\
وفي قَوْلِ المَرَّارِ يَصِفُ نَاقَةً :<br>\
إِذا هِيَ خَرَّتْ خَرَّ مِنْ عَنْ يَمِينِهَا<br>\
شَعِيبٌ بِهِ إِجْمَامُهَا ولُغُوبُهَا<br>\
يَعْنِي الرَّحْلَ ؛ لأَنَّه مَشْعُوبٌ بَعْضُهُ إِلَى بَعْضٍ أَي مَضُمُومٌ .<br>\
( والشُّعْبَةُ بالضَّمِّ : ما بَيْنَ القَرْنَيْن ) لتَفْرِيقِهِما بَيْنَهُمَا ( و ) ما بَيْن ( الغُصْنَيْن ) ومثله في الأَساس .<br>\
( و ) الشُّعْبَة : الفِرْقَةُ و ( الطَّائِفَةُ من الشَّيْءِ ) .<br>\
وفي يَدِه شُعْبَةُ خَيْرٍ مَثَلٌ بِذلِكَ .<br>\
ويقال : الشْعَبْ لِي شُعْبَةً مِنَ الْمَالِ أَي أَعْطِنِي قِطْعَةً مِنْ مَالِك .<br>\
وفي يَدِي شُعْبَةٌ مِنْ مَالٍ .<br>\
وفي الحديث : ( الحَيَاءُ شُعْبَةٌ مِن الإِيمان ) أَي طَائِفَةٌ مِنْه وقِطْعَة .<br>\
وَفي حَدِيثِ ابْنِ مَسْعُود : ( الشَّبَابُ ) شُعْبَةٌ من الجُنُونِ ) وقَوْلُه تَعَالَى : { إِلَى ظِلّ ذِى ثَلَاثِ شُعَبٍ } ( المرسلات : 30 ) .<br>\
قال ثَعْلَب : يقال : إِنَّ النارَ يَوْمَ القِيَامة تنفَرِق ( إِلى ) ثَلَاث فِرَق فكُلَّمَا<br>\
ذَهَبُوا أَنْ يَخْرُجُوا إِلى مَوْضِعٍ رَدَّتْهُم .<br>\
ومعنى الظِّلِّ هُنَا أَنَّ النَّارَ أَظَلَّتْه لِأَنَّه لَيْسَ هنَاك ظِلّ ، كذا في لسان العَرَب .<br>\
( و ) الشُّعْبَةُ من الشَّجَرِ : ما تَفَرَّقَ مِنْ أَغْصَانِها .<br>\
قَالَ لَبِيدٌ :<br>\
تَسْلُبُ الكَانِسَ لم يُؤْرَها<br>\
شُعْبَة السَّاقِ إِذا الظِّلُّ عَقَلْ<br>\
وتَشَعَّبَ أَغْصَانُ الشَّجَرَة وانْشَعَبَت : انْتَشَرت وتَفَرَّقَتْ .<br>\
وشُعْبَة : غُصْن من أَغْصَانِها وقيل : الشُّعْبةُ : ( طَرَفُ الغُصْنِ ) ، وَهُوَ مَجَاز .<br>\
وشُعَبُهُ : أَطْرَافُه المُتَفَرِّقَةُ ، وكُلُّه رَاجِع إِلىَ مَعْنَى الافْتِرَاقِ ، وقِيلَ : مَا بَيْنَ كُلِّ غُصْنَيْن شُعْبَة .<br>\
ويُقَالُ : هَذِه عَصا فِي رَأْسِهَا شُعْبَتَان .<br>\
قال الأَزْهَرِيّ : وسَمَاعِي مِن الْعَرَبِ عَصا في رَأْسِها شُعْبَان ، وبغَيْر تَاءٍ ، كَذَا قَالَه ابْنُ مَنْظُور .<br>\
وفي الأَسَاسِ ، ومِنَ المَجَازِ : أَنَا شُعْبَةٌ مِنْ دَوْحَتِكَ وغُصْنٌ مِنْ سَرْحَتِك .<br>\
( و ) الشُّعْبَةُ : ( المَسِيلُ في ) ارتِفَاع قَرَارَة ( الرَّمْلِ ) .<br>\
والشُّعْبَةُ : المَسِيلُ الصَّغيرُ .<br>\
يقال : شُعْبَ حافِلٌ أَي مُمْتَلِئَةٌ سَيْلاً .<br>\
( و ) الشُّعْبَةُ : ( مَا صَغُرَ مِنَ ) وفي نسخة عَنِ ( التَّلعَةِ ) .<br>\
( و ) قِيلَ : ( ما عَظُمَ من سوَاقِي الأَوْدِيَةِ ) .<br>\
وقِيلَ : الشُّعْبَةُ : ما انْشَعَبَ من التَّلْعَةِ والوَادِي أَي عَدَلَ عَنه وأَخَذَ في طَرِيقٍ غَيرِ طَرِيقِه فتِلْكَ الشُّعْبَةُ .<br>\
( و ) الشُّعْبَةُ : ( صَدْعٌ في الجَبَلِ يَأْوِي إِلي المَطَرُ ) ، كذا في النُّسَخِ وصَوَابُه الطَّيْرُ ، كَذَا في لِسَانِ العَرَب وزَادَ وَهُوَ مِنْهُ .<br>\
( ج ) أَي جَمْعُ الكلِّ ( شُعَبٌ وشِعَابٌ ) الشُّعْبَة : دون الشِّعْبِ .<br>\
( و ) من المَجَاز : ( شُعَبُ الفَرَسِ ) وأَقْطَارُه : ( نَوَاحِيهِ كُلُّهَا ) .<br>\
قال دُكَيْنُ بْنُ راءٍ :<br>\
أَشَم خِنْذِيذٌ مُنِيفٌ شُعَبُهْ<br>\
يَقْتَحِمُ الفَارِسَ لولا قَيْقَبُهْ .<br>\
( أَو ) الشُّعَبُ : ( ما أَشْرَفَ مِنْهَا ) أَي نَواحِيه .<br>\
وفي بَعْضِ النُّسَخ مِنْهُ ، فالضَّميرُ للْفَرَس ، المُرَادُ بما أَشْرَفَ مِنْه كالعُنُق والمَنْسِج والحَجَبَات .<br>\
وشُعَبُ الدَّهْرِ : حالَاتُه ، قَالُه اللَّيْث .<br>\
وأَنْشَدَ قَوْلَ ذِي الرُّمَّة المُتَقَدِّم الَّذِي هُوَ :<br>\
ولا تَقَسَّمُ شَعْباً واحداً شُعَ<br>\
وفَسَّرَهُ فَقَال : أَي ظَنَنْتُ أَن لا يَنْقَسِم الأَمْرُ الوَاحِدُ إِلَى أُمُورٍ كَثِيرَة .<br>\
قال الأَزْهَرِيُّ : ولم يجوِّد الليثُ في تَفْسِير البَيْتِ ، ومَعناهُ أَنَّه وَصَفَ أحْيَاءً كانوا مُجْتَمِعِين في الرَّبِيعِ ، فلما قَصَدُوا المَحَاضِرَ تَقَسَّمَتْهُم المِيَاهُ .<br>\
وشُعَبُ القَوْمِ : نِيَّاتُهُم في هَذَا البَيْت ، وكانت لِكُلِّ فِرْقَةٍ مِنْهُم نِيَّةٌ غيرُ نِيَّةِ الآخَرِينٍ فَقَالَ : ما كنتُ أَظُنُّ أَنَّ نِيَّاتٍ مُخْتَلِفةً تُفَرِّقُ نِيَّةً مُجْتَمِعَةً ، وذَلِكَ أَنَّهم كَانُوا في مُنْتَوَاهُم ومُنْتَجَعِهم مُجْتَمِعِين عَلَى نِيَّةٍ وَاحِدَة ، فلما هَاجَ العُشْبُ ونَشَّتِ الغُدْرَانُ تَوَزَّعَتْهم المَحَاضِرُ وأَعْدَادُ المِيَاهِ ، فهذا مَعْنَى قَوْله :<br>\
ولا تَقَسَّمُ شَعْباً واحِداً شُعَبُ<br>\
انْتَهَى من لِسَانِ العَرب .<br>\
ومن المجاز : نُوَبُ الزَّمَانِ وشُعَبُه : حَالَاتُه ، كَذَا فِي الأَسَاسِ .<br>\
( وَشَعُوبُ : قَبِيلَةٌ ) .<br>\
قال أَبو خِرَاش :<br>\
مَنَعْنَا من عَدِيِّ بني حُنَيْفٍ<br>\
صِحَابَ مُضَرِّسٍ وابْنَىْ شَعُوبا<br>\
فأَثْنِوا يا بَنِي شِجْعٍ عَلَيْنَا<br>\
وحَقُّ ابْنَيْ شَعُوبٍ أَن يُثِيبَا<br>\
قال ابنُ سِيدَه : كَذَا وَجَدْنَا شَعُوبٍ مَصُرُوفاً في البَيْتِ الأَخِير .<br>\
ولو لم يُصْرَف لاحْتَمَل الزِّحَافَ .<br>\
( و ) شَعُوبُ : اسْمُ ( المَنِيَّةِ ) ، ذَكَرَهُ غَيْرُ وَاحِدٍ بِغَيْره أَلِفٍ ولَامٍ ( كالشَّعُوب ) مَعْرِفةً ، وقد أَنْكَرَه جَمَاعَةٌ وعَدُّوه مِنَ اللَّحْنِ .<br>\
وفي الصَّحَاح : الشُّعْبَةُ : الفرْقَةُ .<br>\
تقُولُ : شَعَبَتْهُم المَنِيَّةُ أَي فَرَّقَتْهُم ، ومِنْه : سُمِّيَتِ المَنِيَّةُ شَعُوبَ ، وهي مَعْرِفَةٌ لا تَنْصَرِفُ ولا يَدْخُلُها الأَلِفُ والَّلامُ .<br>\
وفي لِسَانِ العَرَبِ : وقِيلَ : شَعُوبُ<br>\
والشَّعُوبُ كِلْتَاهُمَا المَنِيَّة لأَنَّه تُفَرِّقُ .<br>\
أَما قَوْلُهم فيها شَعُوبُ ، بغير لام ، والشَّعُوبُ ، باللام ، فقد يُمْكِن أَن يَكُونَ في الأَصْل صِفَةً لأَنَّه من أَمْثِلَة الصَّفَاتِ بِمَنْزِلَة قَتُول وضَرُوب ، وإِذَا كَانَ كَذَلِك فَالَّلام فِيه بمَنْزِلَتها في العَبَّاسِ والْحَسَن والحَارِث .<br>\
ويُؤَكِّد هَذَا عِنْدَك أَنَّهم قَالُوا في اشْتِقَاقِها إنَّمَا سُمِّيت شَعُوب لأَنَّها تَشْعَبُ أَي تُفَرِّقُ وهذا المَعْنَى يُؤَكِّد الوَصْفِيَّة فِيها ، وهَذَا أَقْوَى من أَن تُجْعَلَ الَّلامُ زَائِدَةً .<br>\
ومن قَالَ شَعُوبُ ، بِلَا لَامٍ ، خَلَصَتْ عِنْدَه اسْماً صَرِيحاً ، وأَعْرَاهَا في اللَّفْظِ من مَذْهَبِ الصَّفَة ، فلِذَلِكَ لم يُلْزِمْهَا الَّلام كما فَعَلَ ذَلِكَ مَنْ قَالَ : عَبَّاس وحَارِث إلَّا أَنَّ رَوَائِح الصَّفَةِ فِيهِ عَلَى كُلِّ حَالٍ وإِنْ لم تَكُنْ فيه لَامٌ ، أَلَا تَرَى أَنَّ أَبَا زَيْد حَكَى أَنهم يُسَمونَ الخُبْزَ جَابِرَ بْنَ حَبَّةَ ؛ وإِنَّما سَمَّوْه بِذَلِك لأَنه يَجْبُر الجَائِع ، فَقَدْ تَرَى مَعْنَى الصَّفَة فِيه وإِنْ لَمْ تَدْخُلْه الَّلامُ .<br>\
ومِنْ ذَلِكَ قَوْلُهم : وَاسِطٌ قال سِيبَوَيْهِ : سَمَّوْهُ واسِطاً ؛ لأَنَّه وَسَطَ بَيْنَ العِرَاقِ والبَصْرَةِ ، فمَعْنَى الصِّفَةِ فِيهِ وإِن لمْ يَكُن في لَفْظِه لَامٌ ، انْتَهَى .<br>\
ويُقَالُ : أَقَصَّتُه شَعُوبُ إِقْصَاصاً إِذَ أَشْرَف على المَنِيَّة ثم نَجَا .<br>\
وفي حَدِيثِ طَلْحَةَ : ( فما زِلْتُ وَاضِعاً رِجْلِي عَلَى خَدِّه حَتَّى أَزَرْتُه شَعُوبَ ) أَي المَنِيَّة .<br>\
وأَزَرْتُه مِنَ الزِّيَارَة .<br>\
وقَال نَافِعُ بْنُ لَقِيطٍ الأَسَدِيُّ :<br>\
ذَهَبَتْ شَعُوبُ بأَهْلِهِ وبِمَالِه<br>\
إِن المَنَايا للِرِّجَالِ شَعُوبُ<br>\
( و ) شَعُوبُ : ( ع باليَمَن ) .<br>\
وفي التكملة قَصْرٌ باليَمَن .<br>\
( وشَعَبَ كمنَعَ : ظَهَرَ ) ، ومِنْه سُمِيّ الشّهْرُ كَمَا سَيَأْتِي .<br>\
( و ) شَعَبَ ( البَعِيرُ ) : يَشْعَب شَعْباً ( اهْتَضَم الشجرَ مِنْ أَعْلَاهُ ) .<br>\
قال ثَعْلَبٌ : قَالَ النَّضْر بنُ شُمَيْل : سَمِعْتُ أَعْرَابِيًّا حِجَازِيًّا بَاعَ بَعِيراً لَهُ يَقُولُ : أَبِيعُك هو يَشْبَعُ عَرْضاً وشَعْباً .<br>\
العَرْضُ : أَن يَتَنَاوَلَ الشَّجَر مِنْ أَعْرَاضِهِ .<br>\
( و ) شَعَبَ ( فُلَاناً : شَغَلَه ) .<br>\
يقال : ما شَعَبَكَ عَنِّي ، أَي مَا شَغَلَك .<br>\
( و ) شَعَبَ الأَمِيرُ رَسُولاً إِلَيْه : أَرْسَلَه ) .<br>\
( و ) شَعَبَ ( اللِّجَامُ الْفَرَسَ ) إِذا ( كَفَّه عَنْ جِهَةِ قَصْدِهِ ) ولم يَدَعْه يَمْضِي عَلَى جِهَتِه .<br>\
قال دُكَين :<br>\
شَاحِيَ فِيهِ واللِّجَامُ يَشْعَبُه<br>\
وفي الشِّمَالِ سَوْطُه ومِخْلَبُه<br>\
( و ) شَعَبَه يَشْعَبُه شَعْباً إِذَا ( صَرَفَه ) .<br>\
( و ) شَعَبَ ( إِلَيْهِم ) في عَدَدِ كَذَا : ( نَزَعَ وفَارَق صَحْبَه ) .<br>\
( وشَعْبَانُ : قَبِيلَة .<br>\
و : ع بالشّأم ) .<br>\
في لسان العرب : شَعْبَان : بَطْنٌ من هَمْدَان تَشَعَّب مِن الحيَمَن .<br>\
إِلَيْهم يُنْسَبُ عَامِ الشَّعْبِي على طَرحِ الزَّائد .<br>\
وقد تَقَدَّم أَنَّ مَنْ نَزَلَ الشَّأم مِنْ وَلَدِ حَسَّانَ بْنِ عَمْرو الحِمْيَرِيِّ يُقَال لهن : الشَّعْبَانِيُّون .<br>\
