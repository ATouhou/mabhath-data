( و ) الشَّرَبَة ( بالتحريك : كَثْرَةُ الشُّرْبِ ) وجَمْعُ شَارِب كَكَتَبَة جَمْع كَاتِب ، نَقَلَه الفَيُّومِيّ في المِصْبَاحِ .<br>\
قال أَبو حَنِيفَة : قَالَ أَبُو عَمْرو : إِنَّه لَذو شَرَبَة إِذَا كَانَ كَثِيرَ الشُّرْب .<br>\
( و ) الشَّرَبَةُ مِثْلُ ( الحُوَيْضِ ) يُحْفَر ( حَوْلَ النَّخْلَة ) والشَّجَرة يُمْلَأُ مَاءً ( يَسَعُ رِيَّها ) فتَترَوَّى منه .<br>\
والجَمع شَرَبٌ وشَرَبَاتٌ .<br>\
قال زُهَيْرٌ :<br>\
يَخْرُجْنَ مِنْ شَرَبَاتٍ مَاؤُهَا طَحِلٌ<br>\
عَلَى الجُذُوعِ يَخُفْنَ الغَمَّ والغَرَقَا<br>\
وأَنْشَدَ ابْنُ الأَعْرَابِيّ : .<br>\
مِثْل النَّخِيل يُرَوِّي فَرْعَها الشَّرَبُ :<br>\
وفي حَدِيث عُمَرَ رَضِي الله عَنْه ( اذْهَبْ إِلَى شَرَبَةٍ من الشَّرَبَات فادلُك رَأْسَك حتى تُنَقِّيَه ) وفي حَدِيثِ جَابِر : ( أَتَانَا رَسُولُ اللّهِ صَلَّى اللّهُ عَليْهِ وسَلَّم فَعَدَلَ إِلَى الرَّبِيعِ فَتطَهَّر وأَقْبل إِلى الشَّرَبَّة ) .<br>\
الرَّبِيعُ : النهر .<br>\
( و ) الشَّرَبَ : ( كُرْدُ الدَّبْرَة ) ، وهي المِسْقَاةُ .<br>\
والجَمْعُ مِنْ ذلِك كُلِّه شَرَبَاتٌ وشَرَبٌ .<br>\
( و ) الشَّرَبَةُ : ( العَطَشُ ) .<br>\
ولم تَزَلْ بِهِ شَرَبَةٌ ( هذا ) اليومَ أَيْ عَطَشٌ ، قاله اللِّحْيَانيّ .<br>\
وفي التهذيب : جَاءَتِ الإِبِلُ وبِهَا شَرَبَةٌ أَي عَطَشٌ .<br>\
وقد اشْتَدَّت شَرَبَتُها .<br>\
.<br>\
وطَعَامٌ مَشْرَبَةٌ : يُشْرَبُ عَلَيْهِ الماءُ كَثِيراً .<br>\
وطَعَامٌ ذُ شَرَبة إِذَا كانَ لا يُرْوَى فِيه من الماءِ .<br>\
وفي لسن العرب : الشَّرَبَةُ : عَطَشُ المَالِ بَعْدَ الجَزْءِ ؛ لأَنَّ ذلِك يَدْعُوهَا إِلَى الشُّرْب .<br>\
( و ) الشَّرَبَةُ : ( شِدَّةُ الحَرِّ ) .<br>\
يقال : يَوْمٌ ذو شَرَبَةٍ أَي شَدِيدُ الحَرِّ يُشْرَبُ فِيهِ المَاءُ أَكثر مِمّا يُشْرَبُ في غيره .<br>\
( والشَّوَارِبُ عُرُوقٌ في الحَلْقِ ) تَشْرَبُ المَاءَ ، وهي مَجَارِيه ، وقيل : هي عُرُوقٌ لازِقَةٌ بالحُلْقُومِ وأَسْفَلُهَا بالرِّئَة ، قَالَه ابْنُ دُرَيْدِ .<br>\
ويقال : بَلْ مُؤَخَّرُهَا إِلَى الوَتِين ، ولَهَا قَصَب منه يَخْرُجُ الصَّوْت .<br>\
.<br>\
( و ) قيل : هي ( مَجَارِي الماءِ في العُنُقِ ) وَهِيَ الَّتي يَقَعُ فِيهَا الشَّرَقُ ومِنْهَا يَخْرُج الرِّيقُ ، وقيل : شَوَارِبُ الفَرَسِ : نَاحِيَةُ أَوْدَاجه حَيْثُ يُوَدِّجُ البَيْطَارُ ، وَاحِدُهَا في التَّقْدِيرِ شَارِبٌ .<br>\
وحِمَارٌ صَخِبُ الشَّوَارِبِ ، مِنْ هَذَا ، أَي شَدِيدُ النَّهِيقِ .<br>\
وفي الأَسَاس ، ومِن المَجَازِ : يُقَال للمُنْكَرِ الصَّوْتِ : صَخِبُ الشَّوَارِبِ ،<br>\
يُشَبَّه بالحِمَارِ ، انتهى .<br>\
وفي لِسَانِ العَرَبَ عَنِ ابْنِ الأَعْرَبِيّ : الشَّوَارِبُ : مَجَارِي المَاءِ في العَيْنِ .<br>\
قال أَبُو مَنْصُور : أَحْسَبُه ( أَراد ) مَجارِيَ المَاءِ في العينِ الَّتِي تَفُورُ في الأَرْضِ لا مَجَارِيَ مَاءِ عَيْنِ الرَّأْسِ .<br>\
( و ) الشَّوَارِبُ : ( مَا سَالَ على الفَمِ من الشَّعَر ) .<br>\
قال اللِّحْيَانِيُّ : وقَالُوا : إِنَّه لَعَظِيمُ الشَّوَارِبِ ، قال : وَهُوَ من الوَاحِدِ ( الذي ) فُرِّقَ فجُعِلَ كُلُّ جُزْءٍ منه شَارِباً ، ثم جُمِعَ عَلَى هَذا .<br>\
وقد طَرَّ شَارِب الغُلَامِ ، وهُمَا شَارِبَانِ ، انْتَهَى .<br>\
وقيل : إِنَّمَا هُو الشَّارِبُ والتَّثْنِيَة خَطَأٌ .<br>\
وقال أَبُو عَلِيّ الفَارِسيّ : لا يَكَاد الشَّارِب يُثَنَّى ، ومثلُه قَوْلُ أَبِي حَاتِم .<br>\
وقال أَبُو عُبَيْدَةَ : قالَ الكِلَابِيُّون : شَارِبَانِ بِاعْتِبَارِ الطَّرَفَيْن الجَمْعُ شَوَارِب ، نَقَلَه شَيْخُنا .<br>\
وأَنْشَدَنِي الأَدِيبُ المَاهِر حَسَن بْنُ مُحَمَّد المَنْصُورِيّ بدَجْوَة مِنْ لَطَائِف ابْنِ نُبَاتَة :<br>\
لقد كنْتَ لي وَحْدِي وَوَجْهُك جَنَّتي<br>\
وكُنَّا وكَانَت للِزَّمَانِ مَوَاهِبُ<br>\
فعَارَضَنِي في رَوْض خَدِّك عَارضٌ<br>\
وزَاحَمَني في وِرْدِ رِيقِك شَارِبُ<br>\
( و ) الشَّارِبَان عَلَى مَا فِي التَّهْذِيب وغَيْرِه : ( ما طَالَ مِنْ نَاحِيَةِ السَّبَلَةِ ، أَو السَّبَلَةُ كُلُّها شَارِب ) واحد .<br>\
قَالَه بَعْضُهُ ، ولَيْسَ بِصَوَابٍ .<br>\
( و ) مِنَ المَجَازِ : ( أُشْرِبَ فلانٌ حُبَّ فُلَانٍ ) كذَا في النَّسَخ .<br>\
وفي غير وَاحِدٍ من الأُمَّهَاتِ ( فُلَانَةَ ( أَي خَالَطَ قَلْبَه ) .<br>\
وأُشْرِبَ قَلْبُه مَحَبَّةَ هذَا ، أَيْ حَلَّ مَحَلَّ الشَّرَابِ .<br>\
وفي التَّنْزِيلِ : { وَأُشْرِبُواْ فِى قُلُوبِهِمُ الْعِجْلَ } ( البقرة : 93 ) أَي حُبَّ العِجْلِ ، فَحُذِفَ المُضَافُ وأُقِيم المُضَافُ إِلَيْه مُقَامه ، ولا يَجُوزُ أَنْ يَكُونَ العِجْلُ هو المُشْرَب لأَنَّ العِجْلَ لا يُشْرَبُه القَلْبُ .<br>\
وقال الزَّجَّاج : معناه أَي سُقُوا حُبَّ العِجْلِ ، فَحُذِفَ حُبّ وأُقِيم العِجْلِ مُقَامَه ، كما قَالَ الشَّاعِر :<br>\
وكَيْفَ تُوَاصِلُ مَنْ أَصْبَحَتْ<br>\
خَلَالَتُه كَأَبِي مَرْحَبِ<br>\
أَي كخَلَالِة أَبِي مَرْحَبٍ .<br>\
وأُشْرِبَ قَلْبُه كَذَا أَي حَلَّ مَحَلَّ الشَّرَابِ أَوِ اخْتَلَطَ بِه كما يَخْتَلِط الصِّبْغُ بالثَّوْبِ .<br>\
وفي حديث أَبي بكر : ( وأُشْرِب قلبُه الإِشْفَاقَ ) كذا في لسان لعرب .<br>\
وفي الأَسَسِ ، ومِنَ المَجَازِ : قَوْلُهم : رَفَع يعدَه فأَشْرَبَهَا الهَوَاءَ ثم قَالَ بها على قَذَالى .<br>\
( و ) مِنَ المَجَازِ : ( تَشَرَّبَ ) الصِّبْغُ في الثَّوُب : ( سَرَى ) ، والصِّبْغُ يَتَشَرَّبُ الثَّوْبَ .<br>\
( و ) تَشَرَّبَ ( الثوبُ العَرَقَ : نَشِفَهُ ) ، هكذَا في نُسْخَتِنَا .<br>\
والَّذِي في الأَسَاسِ ولِسَانِ العَرَب : الثَّوْبُ يَتَشَرّب الصِّبّغ أَي يَتَنَشَّفُه ، والثوبُ يَشْرَبُ الصِّبْغَ يَنْشَفُه .<br>\
( واسْتَشْرَبَ لَوْنُه : اشْتَدَّ ) .<br>\
يقال : اسْتَشْرَبَتِ القوسُ حُمْرَةً أَي اشْتَدَّت حُمْرَتُها ، وذلك إِذَا كَانَت مِنَ الشِّرْبانِ ، حكاه أَبو حنيفة .<br>\
( والمَشْرَبَةُ ) بالفتح في الأَوَّل والثَّالِثِ ، ( وتُضَمُّ الرَّاء : أَرْضٌ لَيِّنَةٌ دَائِمَةُ النَّبَاتِ ) أَي لا يزال فِيهَا نَبْتٌ أَخْضَرُ رَيَّانُ .<br>\
( و ) المَشْرَبَةُ ، بالوَجْهَيْنِ : ( الغُرْفَةُ ) ، قال في الأَسَاس : لأَنَّهم يَشْرَبُون فِيهَا .<br>\
وعن سيبويه : جَعَلُوه اسْماً كالغُرْفَة .<br>\
وفي الحَدِيثِ ( أَنَّ النَّبيَّ صَلَى الله عليه وسَلَم كَانَ في مَشْرُبَة لَه ) أَي كَانَ في غُرْفَة وجَمْعُها مَشْرَبَاتٌ ومَشَارِبُ .<br>\
( و ) المَشْرَبَةُ : ( العِلِّيَّةُ ) .<br>\
قال شيخنا : هي كعطْفِ التَّفْسير على الغُرْفَة ، وهي أَشْهَرُ من العِلِّيَّة ، وعليه اقْتَصَر الفَيُّوميّ ، انتهى .<br>\
المَشارِبُ : العَلَالِيّ في شِعْر الأَعْشَى .<br>\
( و ) المَشْرَبَةُ ( الصُّفَّةُ ) ، وقِيلَ : هي كالصُّفَّةِ بَيْن يَدَيِ الغُرْفَة .<br>\
( و ) المَشْرَبَةُ : ( المَشْرَعَةُ ) .<br>\
وفي الحديث : ( مَلْعُونٌ مَلْعُونٌ مَنْ أَحَاطَ على مَشْرَبَة ) .<br>\
هي بفَتْح ( الراء ) من غير ضَمَ : الَوْضِعُ الذي يشْرَبُ مِنْه كالمَشْرَعَة ، ويُرِيدُ بالإِحَاطة تَمَلُّكَه ومنْعَ غيره ( منه ) .<br>\
كذا في لسان العرب .<br>\
ويوجد هنا في بعض النسخ بدل المشرعة المِشْرَبة ، كأَنَّه يقول : والمَشْرَبة بالفَتْح وكمكْنَسَة أَي بالكسر ،<br>\
وهو خطأٌ لما عَرَفت .<br>\
وقد يُرَدُّ على المُصَنِّف بوَجْهَيْن : أَوَّ أَنَّ المَشْرَبَة بالوَجْهَين إِنَّمَا هُوَ في مَعْنَى الغُرْفَة فقط ، وبمعنى أَرْض لَيِّنة وَجهٌ وَاحِد وهو الفَتْح ، صَرَّحَ به غَيْرُ وَاحِدٍ .<br>\
وثانِياً أَن المَشْرَبَة بالمَعْنَيَيْن الأَخِيرين إِنَمَّا هو كالصُّفَّة وكالمَشْرَعَة لا هما بِنَفْسِهما كَمَا أَشَرْنَا إِلَى ذلِكَ ، وقد أُغْفِل عن ذلك شَيْخُنا .<br>\
( و ) المِشْرَبَةُ ( كمِكْنَسَة ) وجوَّز شيخُنا فيه الفَتح ، ونَقْلَه عن الفَيُّوميّ : ( الإِنَاءُ يُشْرَبُ فيه ) .<br>\
( والشَّرُوبُ : الَّتِي تَشْتَهِي الفَحْلَ ) .<br>\
يقال : ضَبَّةٌ شَرُوبٌ إِذا كَانَت كَذلِكَ .<br>\
( و ) عنَ أَبي عبيد : شَرَّبَ تَشْرِيباً .<br>\
( تَشْرِيبُ القِرْبَةِ : تَطْييبُهَا بالطِّينِ ) وذَلكَ إِذَا كَانت جَدِيدَةً ، فَجَعَلَ فِيهَا طِيناً وماءً يَطِيبَ طَعْمُها ، وفي نسخة تَطْيِينُها بالنُّونِ ، وهو خطأُ .<br>\
( وشَرِب به ) أَي الرَّجُلِ ( كَسَمِعَ وأَشْرَبَ به ) أَيْضاً : ( كَذَبَ عَلَيْهِ ) .<br>\
( و ) مِنَ المَجَازِ : ( أَشْرَبَ إِبِلَه ) إِذَا ( جَعَلَ لكُلِّ جَمَلٍ قَرِيناً ) ، فَيَقُولُ أَحَدُهُم لِنَاقَتِه : لأُشْرِبَنَّكِ الحِبَالَ النُّسوعَ أَي لأَقْرُنَنَّكِ بِهَا .<br>\
( و ) أَشْرَبَ ( الخَيْلَ : جَعَلَ الحِبَالَ في أَعْنَاقها ) .<br>\
وأَنْشَدَ ثَعْلَبٌ :<br>\
وأَشْرَبْتُهَا الأَقْرَانَ حتى أَنَخْتُهَا<br>\
بِقُرْحَ وقد أَلْقَيْنَ كُلَّ جَنِينِ<br>\
( و ) أَشْرَب ( فُلَاناً ) وكذا البَعِيرَ والدَّابَّة ( الحَبْلَ : جَعَلَه ) أَي وَضَعَه ( في عُنُقِه ) .<br>\
( و ) مِنَ المَجَازِ : ( اشْرَأَبَّ إِليه ) ولَهُ اشْرِئْبَاباً : ( مَدَّ عُنُقَه ليَنْظُرَ ، أَو ) هُوَ إِذَا ( ارْتَفَع ) وعَلَا ، وكُلُّ رَافِعٍ رَأْسَه مُشْرَئِبٌّ ، قَالَه أَبُو عُبَيْد .<br>\
( والاسم الشُّرَأْبِيبَةُ ) بالضَّمِّ ( كالطُّمَأْنِينَة ) .<br>\
.<br>\
وقَالَتْ عَائِشَةُ رَضِيَ للّهُ عَنْهَا ( اشرأَبَّ النِّفَاقُ ، وارْتَدَّتِ العَرَبُ ) .<br>\
أَي ارْتَفَع وعَلَا ، وفي حَدِيث : يُنَادِي يَوْمَ القِيَامَة مُنَاد ، يا أَهْلَ الجَنَّةَ ، ويا أَهْلَ النَّار فيَشْرَئِبُّونَ لِصَوْتِه ) أَي يَرْفَعُونَ رُءُوسَهم ليَنْظُرُوا إِلَيْه .<br>\
وكُلُّ رَافِعٍ رَأْسَه مُشْرَئِبٌّ .<br>\
وأَنْشدَ<br>\
لذي الرُّمَّةِ يَصِفُ الظَّبْيَةَ ورَفْعَها رَأْسَهَا :<br>\
ذَكَرْتُكِ أَنْ مَرَّت بِنَا أُمُّ شَادِنٍ<br>\
أَمَامَ المَطَايَا تَشْرَئِبُّ وتَسْنَحُ<br>\
قال ؛ اشْرأَبَّ مَأْخُوذٌ من المَشْرَبَة ، وَهي الغُرْفَةُ ، كذا في لسان العرب .<br>\
( والشَّرَبَّةُ كجَرَبَّة ) قال شَيْخُنَا : وفي بَعْضِ النُّسَخ كخِدَبَّة ، بكسْرِ الخَاءِ المُعْجَمَةِ ، وفي أُخْرَى بالجيم بَدَل الخَاءِ ، وكِلَاهُمَا على غَيْرِ صَوَاب ، وعن كُرَاع : لَيْسَ في الكلام ( فَعَلَّة ) إِلَّا هذَا أَي الشَّرَبَّة ، وزِيد عليه قَوْلُهُم : جَرَبَّة ، وقد ذُكِرَ في مَوْضِعِهِ ( ولا ثَالِثَ لَهُمَا ) بالاستِقْرَاءِ ، وَهِيَ ( الأَرْضُ ) اللُّيِّنَةُ ( المُعْشِبَةُ ) أَي تُنْبتُ العُشْب ( لا شَجَرَ بها ) .<br>\

وإِلَّا فإِنَّا بِالشَّرَبَّة فاللِّوَى<br>\
نُعَقِّرُ أُمَّاتِ الرِّبَاعِ ونَيْسِرُ<br>\
( و ) شَرَبَّة بتَشْدِيدِ البَاءِ بغَيْر تَعْرِيفٍ : ( ع ) قال سَاعِدَةُ بْنُ جُؤَيَّةَ :<br>\
بِشَرَبَّةٍ دَمِثِ الكَثِيبِ بِدُورِه<br>\

وقال : دَمِثُ الكَثِيب ، لأَنَّ الشَّرَبَّةَ مَوْضِعٌ أَو مَكَانٌ ، قاله ابنُ سِيدَه في المُحْكَم .<br>\
وقال الأَصْمَعِيُّ : الشَّرَبَّ بِنَجْد .<br>\

هي فِيما بَيْن نَخْل ومَعْدِن بَنِي سُلَيْم .<br>\

وإِلَى الأَمِيرِ من الشَّرَبَّة واللِّوَى<br>\

( و ) الشَّرعبَّةُ : ( الطَّرِيقَةُ ) كالمَشْرَب يقال : ما زَالَ فُلَانٌ على شَرَبَّةٍ وَاحِدَةٍ أَي على أَمْرٍ وَاحِدٍ .<br>\
( و ) مِنَ المَجَازِ : عَن أَبِي عَمْرو : الشَّرْبُ : الفَهْمُ .<br>\
يُقَالُ : ( شَرَب كَنَصَرَ ) يَشْرُب شَرْباً إِذَا ( فَهِمَ ) وشَرَب مَا أُلْقِي إِلَيْهِ : فَهِمَه .<br>\
ويقال للبَليد : احْلُب ثم اشْرُبْ .<br>\
أَي ابْرُك ثم افهم .<br>\
وحَلَبَ إِذَا بَرَكَ كما تَقَدَّم .<br>\
( و ) شَرِبَ ( كَفَرِحَ ) إِذَا ( عَطِشَ ) .<br>\
وشَرِبَ إِذَا رَوِيَ ، ضدّ .<br>\

( وشُرْبُوبٌ وشُرْبَةٌ بضَمِّهِنّ ) وقد تقدّم ضَبْطُ الأَخِر بالفَتْح أَيْضاً ، وشَرْبَانُ ( بالفَتْح ) ( مَوَاضِعُ ) قد بَيَّنَّا بَعْضَها .<br>\
ونُحِيل البَقِيَّةَ على مُعْجَم يَاقُوت ومَرَاصِدِ الاطِّلَاعِ فإِنَّهُمَا قد اسْتَوْفَيا بيَانَها .<br>\

في التَّهْذِيبِ : الشَّارِبَانِ : مَا طَالَ مِنْ نَاحِيَة السَّبَلَةِ ، وبِذلِكَ سُمِّيَ شَارِبَا السَّيْفِ .<br>\
