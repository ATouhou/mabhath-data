( و ) الحَصِيرُ : ( الطرِيقُ ) ، عن ابْنِ الأَعْرَابِيّ .<br>\
( و ) الحَصِيرُ : ( الماء ) .<br>\
( و ) الحَصِيرُ : ( الصَّفُّ من النَّاسِ وغَيْرِهِم ) .<br>\
( و ) الحصِيرُ : ( وَجْهُ الأَرْضِ ) ، قيل : وبه سُمِّيَ ما يُفْرَشُ على الأَرض حَصِيراً لكَوْنِه يلِي وَجْهَهَا .<br>\
( ج أَحْصِرَةٌ وحُصُرٌ ) ، بضَمَّتَيْن .<br>\
وأَنشدَ ابنُ الأَعْرَابِيّ في الحُصُر جَمْع حَصْيرٍ بمَعْنَى الطَّرِيق :<br>\
لمَّا رأَيْتُ فِجَاجَ البِيدِ قد وَضَحَتْ<br>\
ولاحَ من نُجُدٍ عَادِيَّةٌ حُصُرُ<br>\
وقد تُسَكَّن الصَّادُ تَخْفِيفاً في جَمْع الحَصِير لِمَا يُفْرش ، كما تَقَدَّمَ .<br>\
( و ) الحَصِيرُ : ( فِرِنْدُ السَّيْفِ ) الَّذي تَراه كأَنَّه مَدبُّ النَّمْلِ .<br>\
قال زُهَيْر :<br>\
بِرَجْمٍ كَوَقْعِ الْهُنْدُوَانِيِّ أَخْلَصَ ال<br>\
صَّياقِلُ منه عن حَصِيرٍ وَرَوْنَقِ<br>\
( أَو ) حَصِيرَاه : ( جَانِباه ) .<br>\
( و ) الحَصِيرُ : ( البَخِيلُ ) المُمْسِك ، كالحَصِر ، ككَتِفٍ .<br>\
( و ) الحَصِيرُ : ( الَّذِي لا يَشْرَبُ الشَّرَابَ<br>\
بُخْلاً ) .<br>\
يُقالُ : شَرِبَ القومُ فحَصِرَ عليهم فُلانٌ أَي بَخِل .<br>\
( و ) الحَصِيرُ : ( جَبَلٌ لِجُهَيْنَةَ ) ، وآخرُ في بِلاَد بَنِي كِلاَبٍ ، ( أَو بِبِلادِ غَطَفَانَ ) ، وقيل هو بالضّاد .<br>\
( و ) الحَصِيرُ : ( كُلُّ ما نُسِجَ مِن جَمِيعِ الأَشْيَاءِ ) ، سُمِّيَ به لحَصْرِ بعْضِ طاقَاتِه على بعْضِ ، فهو فعَيل بمعنى مَفْعُول ، وهو أَعمُّ من البارِيَّة .<br>\
( و ) الحَصِيرُ : ( ثٍ وْبٌ مُزَخْرَفٌ ) منقوش ( مُوَشًّى ) حَسَنٌ ، ( إِذَا نُشِر أَخَذَتِ القُلُوبَ مَآخِذُهُ لحُسْنِه ) .<br>\
وفي النِّهَايَةِ : لحُسْن صَنْعَتِه ، وزَادَ المُصَنِّف في البَصَائِرِ : ووَشْيِه .<br>\
قال : وبه فَسّر بعضُهم حَدِيثَ حُذَيحفَةَ في الفِتَن السَّابِقَ ذِكْرُه ، شَبَّه الفِتَنَ بِذالك ، لأَن الفِتْنَة تُزَيِّن وتُزَخْرِف لِلنّاس والعاقِبَة إِلَى غُرُورٍ .<br>\
وأَنشد المُصَنِّفُ في البَصائِر :<br>\
فَلَيْتَ الدَّهْرَ عَادَ لَنَا جَدِيداً<br>\
وعُدْنا مِثْلَنَا زَمَنَ الحَصِير<br>\
أَي زَمَناً كان بعضُنا يُزَخرِف القول لِبَعْضٍ فنَتَوَادُّ عَليْه .<br>\
( و ) الحَصِيرُ : ( الضَّيِّقُ الصَّدْرِ ) ، كالحَصِرِ والحَصُورِ .<br>\
( و ) الحَصِيرُ : ( وَادٍ ) من أَوْدِيَتِهِم .<br>\
( و ) الحَصِيرُ : ( حِصْنٌ باليمن ) من أَبْنِيَةِ مُلُوكِهِم .<br>\
( و ) الحَصِيرُ : ( ماءٌ من مِيَاهِ نَمَلَى ) قُرْب المَدِينَة المُشَرَّفَة ، ويقال فيه بالضَّاد ، وسيأْتي .<br>\
( و ) الحَصِيرَةُ ، بِهَاءٍ : جرِينُ التَّمْرِ ، وهو المَوْضِع الَّذِي يُحْصَر فيه ، وذكَرِ الأَزْهَرِيُّ بالضَّاد ، وسَيَأْتِي .<br>\
( و ) الحَصِيرَةُ : ( اللَّحْمَةُ المُعْتَرِضَة في جَنْبِ الفَرَسِ ) : وهي ما بَيْنَ الكَتِف إِلَى الخَاصِرَة ، ( تَرَاهَا إِذا ضُمِّرَ ) ، ولا يَخْفَى أَنَّ هاذَا مع ما قَبْلَه في الحَصِير ( أَو لَحْمَةَ كَذالِكَ ) تَكرَارٌ مخلٌّ لاخْتِصَارِه البَالِغ .<br>\
( والحارِثُ بْنُ حَصِيرَةَ ) الأَزْدِيُّ ( مُحَدِّث ) ، وهو أَبو النُّعْمَان الكُوفيّ عن عِكْرَمَة مَوْلَى ابْنِ عَبّاس ، وعَنْهُ عَبْدُ الله بنُ نُمَيْرٍ .<br>\
قال الحَافِظ ابنُ حَجَر في تَحْرِير المُشْتَبه : وعَلَى ضَعْفِهِ يُكْتُب حَدِيثُه ، يُؤْمِن بالرَّجْعَة .<br>\
ووَثَّقَه ابنُ مُعِين والنَّسَائِيّ .<br>\
( وذُو الحَصِيرَيْنِ ) : لقب ( عَبْد مَالِكٍ ) ، وفي بَعْضِ النُّسَخِ عَبْدُ المَلِك ( بْن عَبْدِ الأُلَةِ ) ، بضَمّ الْهَمْزَةَ وفَتْحِ اللاَّم المُخَفَّفَة ( كَغُلَةٍ ) ، وإِنّمَا نَبَّه عَلى وَزْنه لئلاَّ يَشْتَبِه على أَحَدٍ أَنَّهُ عَبْدُ الإِلِ ، واحد الآلِهَة ، وإِنَّمَا لُقِّب به لأَنّه ( كَأَن لَهُ حَصِيرانِ ) منْسُوجَانِ ( مِنْ جَرِيد ) النَّخْلِ ( مُقَيَّرَانِ ) أَي مَطْليَّان بالقِير ، وهو الزِّفْت ، ( يَجْعَلُ أَحَدَهُمَا بَيْنَ يَدَيْهِ والآخرَ خَلْفَه ، ويَسُدُّ بنَفْسِهِ بَابَ الطَّرِيق في الجَبل إِذا جَاءَهم عَدُوٌّ ) .<br>\
( والحَصُورُ ) ، كصَبُورٍ : ( النَّاقَةُ الضَّيِّقَة الإِحْلِيلِ ) .<br>\
ووَرَدَ في بَعْضِ الأُصُول الجَيِّدَة : الأَحالِيل ، بالجَمْع .<br>\
وقد حَصَرت ، بالفَتْح ، وأَحْصَرَت .<br>\
( وحَصُرَ ) الإِحْلِيلُ ، ( ككَرُمَ ، و ) حَصِرَ ، مثلَ ( فَرِحَ ، وأُحْصِر ) بالضَّمِّ .<br>\
( و ) الحَصُورُ : ( مَنْ لا يَأْتِي النِّسَاءَ وَهُوَ قَادِرٌ عَلَى ذالِكَ ) ، وإِنَّمَا يَتْركُهُن عِفَّةٌ وزُهْداً ، وهاذا أَبْلَغُ في المدْح ( أَوْ ) هو ( المَمْنُوعُ مِنْهُنَّ ) ، من الحَصْر والإِحْصار أَي المنْع ، ( أَو ) هُوَ ( مَنْ لا يَشْتَهِيهِنّ ولا يَقْرَبُهُنّ ) وهاذا قَولُ ابْنُ الأَعْرًّبِيِّ .<br>\
وقال الأَزهَرِيّ : الحَصُورُ : مَنْ حُصِرَ عن النِّسَاءِ فلا يَسْتَطِيعُهُن ، وقيل : سُمِّيَ في قوله تَعَالَى : { وَسَيّدًا وَحَصُورًا } ( آل عمران : 39 ) لأَنّه حُبِسَ عمّا يَكُونُ مِن الرّجَال .<br>\
وقال المُصَنِّف في البَصَائِر في تَفْسير هاذِه الآية : الحَصُورُ : الّذِي لا يأْتِي النِّسَاءَ إِمَّا مِنَ العُنّة وإِمّا من العِفّة والاجْتِهَاد في إزالَةِ الشَّهْوَة ، والثَّانِي أَظْهَرُ في الآية ؛ لأَنَّ بذالِكَ يَسْتَحِقّ الرَّجُلُ المَحْمَدَة .<br>\
( و ) قِيلَ الحَصُورُ :<br>\
( المَجْبُوبُ ) الذَّكَرِ والأُنْثَيَيْن ؛ وبه فُسِّر حَدِيثُ ( القِبْطيّ الَّذِي أَمَرَ النَّبِيُّ صلى الله عليه وسلمعَلِيًّا بقَتْله ، قال : فرفَعَت الرِّيحُ ثَوْبَه .<br>\
فإِذا هو حصُورٌ ) .<br>\
قالوا : وهاذا أَبلَغ في الحَصَرِ لعَدم آلَةِ النِّكاح .<br>\
وأَمّا العاقِرُ فإِنّه الَّذِي يأْتِيهٌّ ولا يُولَدُ لَه .<br>\
( و ) الحَصُورُ أَيضاً : ( البَخِيلُ ) المُمْسِكُ .<br>\
وقيل : هو الّذِي لا يُنْفِق على النَّدَامَى ، ( كالحَصِر ) ، ككَتِفٍ ، وَقَدْ جاءَ في حدِيثِ ابنِ عَبّاس ( ما رأَيتُ أَحداً أَخْلَقَ للمُلْكِ مِن مُعاوِيَةَ ، كَان النّاسُ يَرِدُونَ مِنْه أَرْجَاءَ وَاد رَحْبٍ ، لَيْسَ مِثْلَ الحَصرِ العَقِص ) .<br>\
يَعْنِي ابْنَ الزُّبَيْرِ .<br>\
الحَصِرُ : البَخِيل .<br>\
والعَقِصُ : المُلْتَوِي الصَّعْبُ الأَخْلاقِ .<br>\
( و ) الحَصُورُ : ( الهَيُوبُ المُحْجِم عن الشَّيْءِ ) ، وهو البَرِمُ أَيضاً ، كما فسَّره السُّهَيْليّ ، وبه فُسِّر بعضُ بَيْت الأَخْطَلِ السَّابِقِ ذِكْرُهُ .<br>\
( وشاربٍ مُرْبِحٍ . ) إِلى آخره .<br>\
( و ) هم مِمَّن يفضِّلون الحَصورَ ، وهو ( الكَاتِمُ للسِّرِّ ) في نَفْسه الحابِسُ له لا يَبوح به ، كالحَصِرِ ، ككَتِف .<br>\
( والحَصْرَاءُ : الرَّتْقَاءُ ) .<br>\
( والحَصَّارُ ، ككَتّانٍ : اسمُ جَمَاعَة ) .<br>\
منهم أَبو جَعْفَرِ بنُ الحَصَّار المُقْرِي وغيره .<br>\
( و ) الحَصَارُ ، ( ككِتاب وسَحَابٍ : وِسَادٌ رْفَعُ مُؤَخَّرُهَا ويُحْشَى مُقَدَّمُهَا ) فيجعل ( كالرَّحْلِ ) ، أَي كآخِرَتِه في رَفْع المُؤَخّر ، وقادِمَتِه ، في حَشْوِ المُقَدّم ، ( يُلْقَى عَلَى البَعِيرِ .<br>\
و ) قيل هو مَرْكَبٌ ( يَرْكَبُ ) به الرَّاضَةُ وقيل : هو كِساءٌ يُطرَح على ظَهْرِه يُكْتَفَلُ به ، ( كالمِحْصَرَةِ ) ، بالكسر .<br>\
( أَوْ هِي ) ، أَي المِحْصَرة ( قَتَبٌ صَغِيرٌ ) يُحصَر به البَعِير ويُلْقَى عَليه أَدَاةُ الرَّاكِب ، كالحِصَار أَيضاً .<br>\
ومنه حَديثُ أَبي بَكْر ( أَنَّ سعداً الأَسلميَّ قال :<br>\
رَأَيتُه بالخَذَواتِ وقد حَلَّ سُفْرَةً مُعَلَّقةً في مُؤَخَّرَةِ الحِصَارِ ) .<br>\
( وبَعِيرٌ محْصُورٌ : عَلَيْهِ ذالِكَ ) ، وقد حَصَرَه يَحْصُرُه ويَحْصِره واحْتَصَرَه وأَحْصَرَه .<br>\
( و ) المَحْصَرَة ، ( بفَتْحِ المِيمِ : الإِشْرَارَةُ يُجَفَّفُ عَلَيْهَا الأَقِطُ ) .<br>\
( وأَحْصَرَهُ المَرَضُ ) : مَنَعَه مِنَ السَّفَرِ أَو حَاجَةِ يُرِيدُهَا ، قَالَ اللَّهُ عزَّ وجلَّ : { فَإِنْ أُحْصِرْتُمْ } ( البقرة : 196 ) وحُصرَ ، في الحَبْس ، أَقْوى من أُحْصِر ، لأَنّ القرآنَ جاءَ بها ، وقد تقدَّم .<br>\
( أَوْ ) أَحصَره المَرَضُ و ( البَوْلُ : جَعَلَه يَحْصُر نَفْسَه ) .<br>\
وأَصْلُ الحَصْر والإِحصارِ الحَبْسُ .<br>\
يقال : حصَرَنِي الشيْءُ وأَحْصَرني ، أَي حَبسَنِي .<br>\
( والمُحْتَصِرُ : الأَسَدُ .<br>\
ومُحَاصَرَةُ العَدُوِّ ، م ) ، أَي مَعْرُوفٌ .<br>\
يقال : حاصَرَهُم العَدُوُّ حِصَاراً ومُحَاصرَةً .<br>\
وبَقِينَا في الحِصَارِ أَيّاماً .<br>\
وحُوصِرُوا مُحَاصَرَةً شَدِيدَةً .<br>\
( وحَصَرَه ) يَحْصُره حَصْراً : ( اسْتَوْعَبَه ) وحَصَّله وأَحاطَ به .<br>\
( و ) حَصَرَ ( القَوْمُ بِفُلان ) حَصْراً : ضَيَّقوا عليه و ( أَحاطُوا به ) .<br>\
ومنه قَوْلُ الهُذَلِيِّ :<br>\
وقَالُوا تَرَكْنَا القَوْمَ قد حَصَرُوا بِهِ<br>\
ولا غَرْوَ أَنْ قَدْ كَانَ ثَمَّ لَحِيمُ<br>\
( و ) قد حَصِرَ على قومه ( كفَرِحَ : بَخِلَ ) .<br>\
وقال شيخنا : وهو مُسْتَدْرِك ، لأَنه ذَكَره في مَعَاني الحصر وفي معاني الحَصُور ، وقد زَعم الاختصارَ البالغَ ، وهاذا تَطْوِيل بالِغ ، ومثله ما بعده .<br>\
( و ) حَصرَ ( عَن المْرْأَةِ : امْتَنَعَ عن إِتْيَانِهَا ) ، أَي مع القُدرَة ، أَو عَجَزَ عنها ، كما تقدّمت الإشارة إليه في ذِكْرِ معاني الحَصورِ .<br>\
( و ) حَصِرَ ( بالسِّرِّ : كَتَمَه ) في نَفْسِه ولم يَبُحْ به ، وهو حَصِرٌ وحَصُورٌ .<br>\
( والحُصْرِيُّ ، بالضَّمِّ ) .<br>\
