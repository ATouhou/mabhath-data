قالَ : وإسْنادُ التَّرائِي إلى النارَيْن مَجازٌ مِن قَوْلهم دارِي تَنْظُر إلى دارِ فلانٍ ، أَي تُقابِلُها .<br>\
( و ) يقالُ : ( هو منِّي مَرْأًىً ومَسْمَعٌ ) ، بالرَّفْع ( ويُنْصَبُ ) ، وهو مِن الظُّروفِ المَخْصوصَةِ التي أُجْرِيَت مُجْرَى غَيْر المَخْصوصَة عنْدَ سِيْبَوَيْه ، قالَ : هو مثل مَناطَ الثُّرَيَّا ودرج السُّيُول ، ( أي ) هو منِّي ( بحيثُ أَراهُ وأَسْمَعُهُ ) .<br>\
وفي الصِّحاحِ : فلانٌ منِّيبمَرْأَى ومَسْمَع ، أَي حيثُ أَراهُ وأَسْمَعُ قَوْلَه .<br>\
( و ) هُم ( رِئاءُ أَلْفٍ ، بالكسْرِ ) : أَي ( زهاؤهُ في رَأْي العَيْنِ ) ، أَي فيمَا تَرى العَيْن .<br>\
( و ) يقالُ : ( جاءَ حِينَ جَنَّ رُؤيٌ ورُؤياً ، مَضْمُومَتَيْنِ ، و ) رَأْىٌ ورَأْياً ، ( مَفْتُوحَتَيْنِ : أَي حِينَ اخْتَلَطَ الظَّلامُ فَلَمْ يَتَرَاءَوْا ) ؛ كذا في المُحْكَم .<br>\
( وارْتَأَيْنا في الأمْرِ وتَراءَيْنا ) هُ : أي : ( نَظَرْناهُ ) .<br>\
وقال الجوهريُّ : ارْتآهُ ارْتِئاءً ، افْتَعَل من الرَّأْيِ والتَّدْبيرِ .<br>\
وقالَ ابنُ الأثيرِ : هو افْتَعَل مِن رُؤْيَةِ القَلْبِ أَو مِن الرَّأْيِ ، ومَعْنى ارْتَأَى فَكَّرَ وتأَنَّى ، اه .<br>\
وأَنْشَدَ الأزْهريُّ :<br>\
أَلا أَيُّها المُرْتَئِي في الأمُورِ<br>\
سيَجْلُو العَمَى عنكَ تِبْيانها ( والَّرأْيُ : الاعْتِقادُ ) ، اسمٌ لا مَصْدَرٌ كما في المُحْكَم .<br>\
وقالَ الَّراغبُ : هو اعْتِقادُ النَّفْسِ أَحَد النَّقِيضَيْن عن غلبةِ الظنِّ ، وعلى هذا قوْلُه عزَّ وجلَّ : { يَرَوْنَهُمْ مِثْلَيْهِمْ رأْيَ العَيْن } ، أَي يظنُّونَهم بحَسَبِ مُقْتَضى مشاهَدَةِ العَيْنِ مَثِلَيهم .<br>\
( ج آراءٌ ) لم يكَسَّر على غَيْرِ ذلك .<br>\
( و ) حكَى الجوهريُّ في جَمْعه : ( أَرآءٌ ) مَقْلُوبٌ .<br>\
( و ) حكَى اللّحْيانيُّ في جَمْعه : ( أَرْيٌ ) كأَرْعٍ ، ( ورُيٌّ ) بالضَّمِّ ( ورِيٌّ ) بالكسْرِ .<br>\
والذي في نَصّ المُحْكم عن اللّحْياني رُئي بالضم والكسر وصحيح عليه .<br>\
( ورَئِيٌّ ، كغَنِيِّ ) ، قالَ الجوهريُّ : هو على فَعِيلٍ مثْل ضَأنٍ وضَئِينٍ .<br>\
قالَ ابنُ الأثيرِ : ( و ) قد تَكرَّر ( في الحديثِ : أَرَأَيْتَكَ وأَرَأَيْتَكُما وأَرَأَيْتَكُم ، وهي كلمةٌ تَقُولُها العَرَبُ ) عنْدَ الاسْتِخْبارِ ( بمَعْنَى أَخْبِرْني وأَخْبِراني وأَخْبِروني ، والتَّاءُ مَفْتُوحَةٌ ) أَبَداٌ ، هذا نَصّ النهايةِ .<br>\
وقالَ الَّراغبُ : يَجْرِي أَرَأَيْتَ بمجْرَى أَخْبِرْني فتَدْخُلُ عليه الكَافُ وتُتْرَكُ التاءُ على حالتِه في التَّثْنِية والجَمْع والتَّأْنِيثِ ويُسَلَّط التَّغْيير على الكافِ دُونَ التاءِ ، قالَ تعالى : { أَرَأَيْتك هذا الذي كرّمت عليَّ } ؛<br>\
{ قل أَرَأَيْتكم إن أَتَاكم عَذابُ اللَّهِ } ؛ { قُل أَرَأَيْتُم ما تَدعون مِن دُون اللَّهِ } ؛ { قل أَرأَيْتُم إن جَعَلَ اللَّهُ عليكم اللّيْلَ سَرْمداً } ؛ كلُّ ذلكَ فيه مَعْنى التَّنْبيه .<br>\
قُلْتُ : وللفرَّاء والزجَّاج وأبي إسْحق هنا كَلامٌ فيه تَحْقيقٌ انْظُرْه في التَّهْذيبِ تَرَكْتُه لطُولِه .<br>\
ثم قالَ ابنُ الأثيرِ : ( وكَذلِكَ ) تكرر ( أَلَمْ تَر إلَى كذا ) ، أَلَمْ تَر إلَى فلانٍ ، وهي ( كَلمةٌ تُقالُ عند التَّعَجُّبِ ) مِن الشَّيءِ ، وعنْدَ تَنْبِيه المُخاطبِ كقوْلِه تعالى : { أَلَمْ تَر إلى رَبِّك كيفَ مَدَّ الظِّلّ } ، { أَلَمْ تَر إلى الذين خَرَجُوا من دِيارهم } ، { أَلَمْ تَر إلى الذين أُوتُوا نَصِيباً مِن الكِتابِ } ، أَي أَلَمْ تَعْجَب بفِعْلِهم ولا يَنْتَه شَأْنُهُم إليك .<br>\
وقالَ الراغبُ : إذا عُدِّي رَأَيْت بإلى اقْتَضَى مَعْنى النَّظَر المُؤَدِّي للاعْتِبارِ ، وقد تقدَّمَ قرِيباً .<br>\
( و ) حكَى اللّحْيانيُّ : ( هو مَرْآةٌ بكذا ) وأنْ يَفْعَل كذا كَمسْعاةٍ : ( أَي مَخْلَقَةٌ ) ، وكذا الاثْنانِ والجَمْع والمُوءَنَّثْ .<br>\
( وأَنا أَرْأَى ) أَنْ أَفْعَل ذلكَ : أَي ( أَخْلَقُ ) وأَجْدَرُ بِه .<br>\
( والرِّئَةُ ) ، كعِدَةٍ : ( مَوْضِعُ النَّفَسِ والرِّيحِ من الحَيوانِ ) .<br>\
قالَ اللَّيْثُ : تُهْمَزُ ولا تُهْمَزُ .<br>\
وقالَ الَّراغبُ : هو العُضْو المُنْتَشِر عن القَلْب .<br>\
وفي الصِّحاحِ : الرِّئَةُ السَّحْرُ ، مَهْموزٌ ، والهاءُ عِوَضٌ مِن الياءِ ، ( ج رِئاتٌ وَرئُونَ ) ، بكَسْرِهما على ما يَطَّرِد في هذا النَّحْو ؛ قالَ الشَّاعِرُ :<br>\
فغِظْنَاهُمُ حتَّى أَتَى الغَيْظُ مِنْهُم<br>\
قُلوباً وأَكْباداً لهُم ورِئِينَا<br>\
قالَ ابنُ سِيدَه : وإنَّما جازَ جَمْع هذا ونحْوه بالواوِ والنونِ لأنَّها أَسْماءٌ مَجْهودَةٌ مُنْتَقَصَة ولا يُكَسَّر هذا الضَّرْب في أَوَّلِيّته ولا في حَدِّ النِّسْبة .<br>\
( ورَآهُ : أَصابَ رِئَتَه ) ؛ نَقَلَهُ الجَوهرِيُّ وابنُ سِيدَه .<br>\
وقالَ الَّراغبُ : ضَرَبَ رِئَتَه .<br>\
( و ) رَأَى ( الرَّايَةَ : رَكَزَها ) في الأرْضِ ؛ ( كأَرْآها ) ، وهذه عن اللّحْياني ، قال ابن سِيدَه : وهمزه عندي على غير قياس ، وإنمَّا حكمه أرييتها ( و ) رَأَى ( الزَّنْدَ أَوْقَدُهُ فَرَأَى هو ) بنفسه أي وقد ، وهذا المطاوعُ عن كُراعٍ .<br>\
( و ) يقالُ : ( أَرَى اللَّهُ بفُلانٍ ) كذا وكذا : ( أَي أَرَى النَّاسَ به العَذابَ والهَلاكَ ) ، ولا يقالُ ذلكَ إلاَّ في الشَّرِّ ؛ قالَهُ شَمِرٌ .<br>\
( و ) قالَ الأَصْمعيُّ : يقالُ : ( رَأْسٌ مُرْأًى ، كَمُضْنًى : طَويلُ الخَطْمِ فيه تَصْويبٌ ) ؛ كذا في المُحْكَم وفي التَّهْذِيبِ : كهَيْئةِ الإِبْرِيقِ وأَنْشَدا لذي الرُّمَّة :<br>\
وجَذْب البُرَى أَمْراسَ نَجْرانَ رُكِّبَتْ<br>\
أَوَاخِيُّها بالمُرْأَياتِ الرَّواجِف ِقال الأزْهريُّ : يَعْني أَوَاخِيَّ الأَمْراسِ ، وهذا مَثَل .<br>\
وقالَ نَصِيرُ : رؤوس مُرْأَياتٌ كأَنَّها قَوارِيرُقالَ ابنُ سِيدَه : وهذا لا أَعْرِف له فعْلاً ولا مادَّةً .<br>\
( و ) في التَّهذيبِ : ( اسْتَرْأَيْتُه ) في الرَّأْي : أَي ( اسْتَشَرْتُه ؛ ورَاءَيْتُه ) ، على فاعَلْته ، وهو يُرائِيهِ ، أَي ( شاوَرْتُه ) ؛ قالَ عِمْرانُ ابنُ حطَّان :<br>\
فإن نَكُنْ نحن شاوَرْناكَ قُلْتَ لنا<br>\
بالنُّصْحِ مِنْكَ لَنَا فِيمَا نُرائِيكا ( وأَرْأَى ) الرَّجُل ( إرْآءً : صارَ ذا عقلٍ ) ورَأْىٍ وتَدْبيرٍ .<br>\
( و ) قالَ الأَزْهريُّ : أَرْأَى إرْآءً ( تَبَيَّنَتْ ) آرَاؤُه ، وهي ( الحَماقَةُ في وجْهِهِ ) ، وهو ( ضِدٌّ ) وفيه نَظَرٌ .<br>\
( و ) أَرْأَى ( نَظَرَ في المِرْآةِ ) .<br>\
وفي التّهْذيبِ تَراءَى مِن المُرَاءَاةِ وهي لُغَةٌ في رَأْرَأ .<br>\
قال : ( و ) أَرْأَى ( صارَ له رَئيٌّ مِن الجِنِّ ) ، وهو التَّابِعُ .<br>\
( و ) أَرْأَى ( عَمِلَ ) صالحاً ( رئاءً وسُمْعَةً ) .<br>\
قالَ : ( و ) أَرْأَى : ( اشْتَكَى رِئَتَهُ .<br>\
( و ) أَرْأَى : ( حَرَّكَ جَفْنَيْهِ ) ، وفي التَّهْذيبِ : بعَيْنَيْه ، ( عند النَّظَرِ ) تَحْرِيكاً كَثيراً ، وهو يُرْئِي بعَيْنهِ ، وهي لُغَةٌ في رأرأ .<br>\
( و ) أَرْأَى ( تَبِعَ رَأْيَ بعضِ الفُقَهاءِ ) في الفقْهِ .<br>\
( و ) أَرْأَى : ( كَثُرَتْ رُآهُ ) زِنَة رُعاهُ ، وهي أَحْلاَمُه ، جمعُ الرُّؤْيا .<br>\
( و ) أَرْأَى ( البَعيرُ : انْتَكَبَ خَطْمُه على حَلْقِه ) ( قالَهُ النَّصْرُ ، فهو مُرْأًى كمُضْنى ، وهنَّ مُرْأَياتٌ ، وقد تقدَّمَ شاهِدُه قريباً .<br>\
( و ) أَرْأَتِ ( الحامِلُ مِن ) ، النَّاقَةِ والشَّاةِ ، ( غَيْرِ الحافِرِ والسَّبُع : رُئيَ في ضَرْعِها الحَمْلُ واسْتُبِينَ ) ؛ وكذا المَرْأَةُ وجَمِيعُ الحَوامِلِ ، ( فهي مُرْءٍ ومُرْئِيَةٌ ) ؛ نقلهُ ابنُ سِيدَه .<br>\
( و ) قالَ اللّحيانيُّ : يقالُ إنَّه لخبيثٌ و ( لا تَر َما ) فلانٌ ولا تَرى ما فلانٌ ، رفْعاً وجَزْماً ، ( و ) إذا قالوا : إنه لخبيثٌ و ( لَمْ تَرَ مَا ) فُلانٌ قالوه بالجَزْم ، وفلانٌ كُلّه بالرَّفْع ؛ وكذا ( وأوَتَرَما ) عن ابن الأعرابيِّ ؛ وكذا ولَوْ تَرَ ما ولَوْ تَرَى ما ؛ كُلُّ ذلكَ<br>\
( بمعْنَى لا سِيَّما ) ؛ حَكَاه كُلّه عن الكِسائي ، كذا في التهْذِيبِ .<br>\
( وذُو الرَّأْيِ ) : لَقَبُ ( العبَّاسِ بنِ عبْدِ المُطَّلبِ ) الهاشِمِيّ ، رضِيَ اللّهُ عنه .<br>\
( و ) أَيْضاً لَقَبُ ( الحُبابِ بنِ المُنْذرِ ) الأنْصارِيّ لُقِّبَ به يَوْم السَّقِيفَة إذ قالَ : أنَا جُذيلُها المُحكَّك وعُذيقُها المُرَجَّب .<br>\
( و ) أَبو عُثْمان ( رَبيعَةُ ) بنُ أَبي عبْدِ الرحمانِ فروخ التِّيمِيّ مَوْلى آلِ المُنْكَدر صاحِبُ ( الرَّأْيِ ) والقائِلُ به ، سَمِعَ أَنَساً والسائِبَ بن يَزِيد ، وهو ( شَيْخُ مالِكٍ ) والثَّوْرِي وشعْبَةَ ، ماتَ سَنَة 136 .<br>\
( وهِلالُ الرَّأْيِ ) بنُ يَحْيَى بنِ مُسْلم البَصْريُّ ( مِن أعْيانِ الحَنَفِيَّة ) كَثيرُ الخَطَأ لا يُحْتج به .<br>\
( وسُرَّ مَنْ رَأَى ) ، بالضمِّ ، وسَرَّ مَنْ رَأَى ، وسَاءَ مَنْ رَأَى ، وسَامَرَّا ، عن ثَعْلَب وابنِ الأنْباري ؛ وهي لُغاتٌ في المَدينَةِ التي بَناها المُعْتصمُ العبَّاسي ، وقد ذُكِرَتْ ( في ( س ر ر ) .<br>\
( وأَصْحابُ الرَّأْيِ ) عنْدَ أَهْلِ الحدِيث هُم : ( أَصْحابُ القِياسِ لأَنَّهُمْ يقولُون برَأْيِهِم فيما لم يَجِدُوا فيه حَديثاً أَو أَثَراً ) ، أَو فيمَا أَشْكلَ عليهم مِن الحدِيثِ ، قالَهُ ابنُ الأثير .<br>\
وأَمَّا عنْدَ غيرِهم فإنَّه يقالُ : فلانٌ من أَهْلِ الرَّأْي إذا كانَ يَرَى رَأْيَ الخوارِجِ ويقولُ بمَذْهَبِهم ؛ ومنه حدِيثُ الأَزْرَق بنِ قَيْس : ( وفِينا رجُلٌ له رَأْيٌ ) .<br>\
وممَّا يُسْتدركُ عليه :<br>\
يقالُ : رَيْتَه على الحَذْفِ ، أَنْشَدَ ثَعْلَب :<br>\
وَجْناء مُقَوَرَّة الأَقْرابِ يَحْسِبُها<br>\
مَنْ لم يَكُنْ قَبْلُ رَاهَا رَأْيَةً جَمَلاوأنا أَرَأُهُ والأَصْلُ أَرْآهُ ، حَذَفُوا<br>\
الهَمْزةَ وأَلْقَوْا حَرَكَتَها على ما قَبْلَها .<br>\
