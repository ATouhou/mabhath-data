ورجلك } ( الإِسراء : 64 ) ، أَي اجْمَعْ عَلَيْهِم وتَوَعَّدْهُمْ بالشَّرِّ ، وقَدْ قُرِىءَ ( واجْلُبْ ) .<br>\
( و ) جَلَّبَ ( عَلَى فَرَسِهِ ) ، كأَجْلَب ( : صَاحَ ) به مِنْ خَلْفِه واسْتَحَثَّهُ للسَّبْق ، قال شيخُنا : وهُوَ مَضْرُوبٌ عليه في النسخة التي بخطِّ المصنّفِ ، وضَرْبُه صَوَابٌ ، لأَنَّه تقدَّم في كَلاَمِه : جَلَّبَ على الفَرَسِ إِذا زَجَرَه ، قلتُ : وفيه تَأَمُّلٌ .<br>\
( و ) قَدْ جَلَبَ ( الجُرْحُ : بَرَأَ يَجْلِبُ ) بالكَسْرِ ( ويَجْلُبُ ) بالضَّمِّ<br>\
( في الكُلِّ ) مما ذُكِرَ ، وأَجْلَبَ الجُرْحُ : مِثْلُه ، كَذَا في ( لسان العرب ) ، وعن الأَصمعيّ : إِذا عَلَتِ القُّرْحَةَ جِلْدَةُ البُرْءِ قِيلَ : جَلَبَ ، وقُرُوحٌ جَوَالِبُ وجُلَّبٌ ، أَي كَسُكَّرٍ وأَنشد :<br>\
عَافَاكَ رَبِّي مِنْ قُرُوحٍ جُلَّبِ<br>\
وفي الأَساس : وجُلَبُ الجُرُوح : قُشُورُهَا .<br>\
( و ) جَلِبَ ( كسَمِعَ ) يَجْلَبُ ( : اجْتَمَعَ ) ومنه في حديث العَقَبَة ( إِنَّكُمْ تُبَايِعُونَ مُحَمَّداً عَلَى أَنْ تُحَارِبُوا العَرَبَ والعَجَمَ مُجْلِبَةً ) أَي مُجْتَمِعِينَ عَلَى الحَرْبِ ، ومنهُمْ مَن رواهُ بالتَّحْتِيَّةِ بَدَلَ المُوَحَّدَةِ ، وسيأْتي .<br>\
( والجُلْبَةُ بالضِّمَّ ) هِيَ ( القِشْرَةُ ) التي ( تَعْلُو الجُرْحَ عِنْدَ البُرْءِ ) ومنه قَوْلُهُم : طَارَتْ جُلْبَةُ الجُرْحِ .<br>\
( و ) الجُلْبَةُ ( : القِطْعَةُ منَ الغَيْم ) يُقَالُ : مَا في السَّمَاءِ جُلْبَةٌ أَي غَيْمٌ يُطَبِّقُهَا ، عن ابنِ الأَعرابيّ وأَنشد :<br>\
إِذا مَا السَّمَاءُ لَمْ تَكُنْ غَيْرَ جُلْبَةِ<br>\
كَجِلْدَةِ بَيْتِ العَنْكَبُوتِ تُنِيرُهَا<br>\
ومَعْنَى تُنِيرُهَا ، أَي كَأَنَّهَا تَنْسِجُهَا بنِيرٍ .<br>\
( و ) الجُلْبَةُ في الجَبَلِ ( : الحِجَارَةُ تَرَاكَمَ بَعْضُهَا عَلى بَعْضِهَا .<br>\
فلم يَبْقَ فيها طِرِيقٌ لِلدَّوَابِّ ) تَأْخُذُ فيه ، قاله الليث ، ( و ) الجُلْبَةُ أَيضاً ( : القِطْعَةُ المتَفَرِّقَةُ ) ليستْ بمتَّصِلة ( مِنَ الكَلاَ ، و ) الجُلْبَة ( : السَّنَة ) الشَّدِيدَة ، و ) الجُلْبَة ( : العِضاهُ ) بكسْرِ العَيْنِ المُهْمَلَةِ ( المُخْضَرَّةُ ) الغَلِيظَةُ عُودُهَا ، والصُّلْبَةُ شَوْكُهَا ( و ) قِيلَ : الجُلْبَةُ ( : شِدَّةُ الزَّمَانِ ) مثْلُ الكُلْبَة : يقالُ : أَصَابَتْنَا جُلْبَةُ الزَّمَانِ ، وكُلْبَةُ الزَّمَانِ ، قال أَوْس بنُ مَغْرَاءَ التَّمِيمِيُّ :<br>\
لاَ يَسْمَحُونَ إِذَا مَا جُلْبَةٌ أَزَمَتْ<br>\
وليْسَ جارُهُمُ فِيها بِمُخْتَارِ<br>\
( و ) الجُلْبَةُ : شِدَّةُ الجُوعِ وقيلَ : الجُلْبَةُ : الشدَّة والجَهْدُ و ( الجُوعُ )<br>\
قال مالكُ بنُ عُوَيْمِرِ بن عثْمَانَ بنِ حُنَيْشٍ الهُذَلِيُّ وهو المُتَنَخِّلُ ، ويُرْوَى لأَبِي ذُؤَيْبٍ والصَّحِيحُ الأَوَّلُ :<br>\
كَأَنَّمَا بَيْنَ لَحْيَيْهِ ولَبَّتِهِ<br>\
مِنْ جُلْبَةِ الجُوعِ جَيَّارٌ وإِرْزِيزُ<br>\
قال ابن بَرّيّ : الجَيَّارُ : حَرَارَةٌ من غَيْظِ يكونُ في الصَّدْرِ ، والإِرْزِيزُ : الرِّعْدَةُ .<br>\
والجَوَالِبُ : الآفَاتُ والشَّدَائِدُ ، وفي الأَساس : ومنَ المَجَازِ : جَلَبَتْه جَوالبُ الدَّهْرِ .<br>\
( و ) الجُلْبَةُ ( : جِلْدَةٌ تُجْعَلُ عَلَى القَتَبِ ، والجُلْبَةُ ) : حَدِيدةٌ تكونُ في الرَّحْلِ ، ( و ) الجُلْبَةُ ( : حَدِيدَةٌ ) صَغِيرَةٌ ( يُرْقعُ بها القَدَحُ ، و ) الجُلْبَةُ ( : العُوذَةُ تُخْرَزُ علَيْهَا جِلْدَةٌ ) ، وجَمْعُهَا الجُلَبُ ، قاله الليث ، وأَنْشَدَ لِعَلْقَمَةَ بنِ عَبَدَةَ يَصِفُ فَرَساً :<br>\
بِغَوْجٍ لَبَانُهُ يُتَمُّ بَرِيمُه<br>\
عَلَى نِفْثِ رَاقٍ خَشْيَةَ العَيْنِ مُجْلِبِ<br>\
والمُجْلِبُ : الذي يَجْعَلُ العُوذَةَ في جِلْدٍ ثم يَخيط ( عليها فيعلّقها ) علَى الفَرَسِ ، والخَيْطُ الذي تُعْقَد عليه العُوذَةُ يُسَمَّى بَرِيماً ( و ) الجُلْبَة ( من السِّكِّينِ : ) التي تَضُمُّ النِّصَابَ علَى الحَدِيدَةِ ، و ) الجُلْبَة ( : الرُّوبَة ) بالضَّمِّ هِيَ خَميرَةُ اللَّبَنِ ( تُصَبُّ على الحَليب ) لِيَتَرَوَّبَ ، ( و ) الجُلْبَة ( : البُقْعَة ) ، يُقَال : إِنَّه لَفِي جُلْبَة صِدْقٍ ، أَيْ في بُقْعَةِ صِدْقٍ ، ( و ) الجخلْبَةُ ( : بَقْلَةٌ ) ، جَمْعُهَا الجُلَبُ .<br>\
( والجَلْبُ ) بالفَتْح ( : الجِنَايَةُ ) على الإِنْسَانِ وقد ( جَلَبَ ) عليه ( كَنَصَرَ ) : جَنَى .<br>\
( و ) الجِلْبُ ، ( بالكَسْرُ ) وبالضَّمِّ .<br>\
كذا في ( لسان العرب ) ( : الرَّحْلُ بِمَا فيهِ ، أَوْ ) جِلْبُ الرَّحْلِ ( : غِطَاؤُهُ ) .<br>\
قاله ثعلب ، وجِلْبُ الرَّحْلِ وجُلْبُه : عِيدَانُه ، قال العجّاج وشَبَّهَ بَعِيرَهُ بِثَوْرٍ وَحْشِيَ رَائِح وقَدْ أَصَابَهُ المَطَرُ :<br>\
عالَيْتَ أَنْسَاعِي وجِلْبَ الكُورِ<br>\
عَلَى سَرَاةِ رائِحٍ مَمْطُورِ<br>\
قال ابن بَرّيّ : والمشهور في رَجزه :<br>\
بل خِلْتُ أَعْلاَقِي وجِلْبَ كُورِي<br>\
أَعْلاَق : جَمْعُ عِلْقٍ ، وهو النَّفِيسُ من كلّ شَيْءٍ ، والأَنْسَاعُ : الحِبَالُ ، وَاحِدُهَا : نِسْعٌ ، والسَّرَاةُ : الظَّهْرُ ، وأَرَادَ بالرَّائِحِ المَمْطُورِ الثَّوْرَ الوَحْشِيَّ .<br>\
وجِلْبُ الرَّحْلِ وجُلْبُه : أَحْنَاؤُه ، ( و ) قِيلَ : جِلْبُهُ وجُلْبُهُ : ( خَشَبُهُ بِلاَ أَنْسَاعٍ وَأَدَاةٍ ) ويُوجَدُ في بَعْضِ النُّسَخ : خَشَبَةٌ بالرَّفْع ، وهو خَطَأٌ .<br>\
( و ) الجُلْبُ ( بالضَّمِّ ويُكْسَرُ : السَّحَابُ ) الذي ( لاَ مَاءَ فيهِ ) وقِيلَ : سَحَابٌ رَقِيقٌ لاَ مَاءَ فيهِ ، ( أَو ) هُوَ السَّحَابُ ( المُعْتَرضُ ) تَرَاهُ ( كأَنَّه جَبَلٌ ) قال تَأَبَّطَ شَرًّا :<br>\
ولَسْتُ بِجُلْبٍ جُلْبِ لَيْلٍ وقِرَّةٍ<br>\
وَلاَ بِصَفاً صَلْدٍ عَنِ الخَيْرِ مَعْزِلِ<br>\
يَقُولُ : لَسْتُ بِرَجُلٍ لاَ نَفْعَ فيهِ ، ومع ذلك فيه أَذًى ، كذلك السَّحَابِ الذي فيه رِيحٌ وقُرٌّ وَلاَ مَطَرَ فِيهِ ، والجَمْعُ أَجْلاَبٌ .<br>\
( و ) الجُلْبُ ( بالضَّمِّ : سَوَادُ الَّليْلِ ) قال جِرانُ العَوْدِ :<br>\
نَظَرْتُ وصُحْبَتِي بِخُنَيْصِراتٍ<br>\
وجُلْبُ الَّليْلِ يَطْرُدُهُ النَّهَارُ<br>\
( و ) الجُلْبُ ( : ع ) مِنْ مَنَازِل حاجِّ صَنْعَاءَ ، علَى طَرِيقِ تِهَامَةَ ، بيْنَ الجَوْنِ وجازَانَ .<br>\
( والجِلْبَابُ ، كَسِرْدَابٍ ، و ) الجِلِبَّابُ ( كَسِنِمَّارٍ ) مثَّلَ به سيبويهِ ولم يُفَسِّرْه أَحدٌ ، قال السيرافيّ : وأَظُنُّه يعْنِي الجِلْبَاب ، وهو يُذَكَّر ويُؤَنَّثُ ( : القَمِيصُ ) مُطْلَقاً ، وخَصَّه بعضُهم بالمُشْتَمِلِ على البدَنِ كُلِّه ، وفَسَّره<br>\
الجوهريُّ بالمِلْحَفَةِ قاله شيخُنا ، والذي في ( لسان العرب ) : الجِلْبَابُ : ثَوْبٌ أَوْسعُ مِنَ الخِمَارِ دُونَ الرِّدَاءِ ، تُغَطِّي به المرْأَةُ رأْسَها وصدْرَها ، ( و ) قيلَ : هو ( ثَوْبٌ واسعٌ للمرأَةِ دُونَ المِلْحَفَةِ ) ، وقيل : هو المِلْحَفَةُ ، قالت جَنُوبُ أُخْتُ عمْرٍ و ذِي الكَلْبِ تَرْثِيهِ :<br>\
تَمْشِي النُّسْورُ إِلَيْهِ وَهْيَ لاَهِيَةٌ<br>\
مَشْيَ العَذَارَى علَيْهِنَّ الجَلاَبِيبُ<br>\
أَيّ أَنَّ النُّسُورَ آمِنَةٌ مِنْه لا تَفْرَقُه لِكَوْنِهِ مَيْتاً ، فهي تَمْشِي إِليه مَشْيَ العَذَارَى ، وأَوَّلُ المَرْثِيَةِ :<br>\
كُلُّ امْرِىءٍ بِطُوَالِ العَيْشِ مكْذُوبُ<br>\
وكُلُّ مَنْ غَالَبَ الأَيَّامَ مغْلُوبُ<br>\
وقال تعالى : { يُدْنِينَ عَلَيْهِنَّ مِن جَلَابِيبِهِنَّ } ( الأَحزاب : 59 ) ، وقيل : هو ما تُغطّي بهِ المرْأَةٌ ( أَو ) هو ( ما تُغطِّي به ثِيابَها مِن فَوْقُ ، كالمِلْحَفةِ ، أَو هو الخِمارُ ) كذا في ( المحكم ) ، ونقلَه ابنُ السكّيت عن العامِريَّة ، وقيل : هو الإِزارُ ، قاله ابنُ الأَعرابيّ ، وقد جاءَ ذِكرُه في حديثِ أُمِّ عَطِيَّةَ ، وقيل : جِلْبابُها : مُلاءَتُها تَشْتمِل بِها ، وقال الخفاجِيُّ في العِناية : قِيل : هو في الأَصْلِ المِلْحَفَةُ ثم اسْتُعِير لِغَيْرِهَا منَ الثِّيَابِ ، ونَقَلَ الحافظُ ابنُ حَجرٍ في المُقَدَّمة عن النَّضْرِ : الجِلْبَابُ : ثَوْبٌ أَقْصرُ مِنَ الخِمَارِ وأَعْرضُ منه ، وهو المِقْنَعَة ، قاله شيخُنا ، والجمْعُ جَلاَبِيبُ ، وقَدْ تَجعلْبَبْتُ ، قال يَصِفُ الشَّيْبَ :<br>\
حتَّى اكْتَسَى الرَّأْسُ قِنَاعاً أَشْهَبَا<br>\
أَكْرَهَ جلْبَابِ لِمَنْ تَجَلْبَبا<br>\
وقال آخَرُ :<br>\
مُجَلْبَبٌ مِنْ سَوَادِ الَّليْلِ جِلْبَابَا<br>\
