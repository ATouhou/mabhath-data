وصَوَابُه بالضَّمِّ كَمَا شَرَحْنَا .<br>\
( و ) الصَّبِيبُ : ( مَاءُ شَجَر السِّمْسِمِ ) .<br>\
وفي حديثِ عُقْبَةَ بْنِ عَامِر ( أَنَّه كَانَ يَخْتَصِبُ بالصَّبِيب ) .<br>\
قال أَبُو عُبَيْدَةَ : يُقَالُ : إِنَّه مَاءُ وَرَقِ السِّمْسِمِ أَوْ غَيْره مِنْ نَبَاتِ الأَرْض .<br>\
قال : وقَدْ وُصِفَ لِي بِمِصر ، ولَوْنُ مَائِه أَحْمَر يَعْلُوه سَوَادٌ .<br>\
وأَنْشَدَ قَوْلَ عَلْقَمَة<br>\
بْنِ عَبَدَةَ السَّابق ذِكْره .<br>\
( و ) الصَّبِيبُ : ( شَيْءٌ كالوَسْمَةِ ) يُخْضَبُ به اللِّحَى .<br>\
( و ) قيل : هو عُصَارَةُ العَنْدَم .<br>\
و ) قيلَ هُوَ ( صِبْغٌ أَحْمَر .<br>\
و ) الصَّبِيبُ أَيْضاً : ( المَاءُ المَصْبُوبُ ) .<br>\
وهَذِه الأَقْوالُ كُلُّهَا بهذا التَّفْصِيلِ في المحكَمِ ولِسَان العَرَبَ وغَيْرِهِمَا مِنْ كُتُبِ الفَنِّ .<br>\
( و ) الصَّبِيبُ : ( العَسَلُ الجَيِّد ) نَقَلَه الصَّاغَانِيّ ، ( وطَرَفُ السَّيْفِ ) ، في قَتْل أَبِي رَافِعٍ اليَهُودِيّ : ( فوضَعْت صَبيبَ السَّيْفِ في بَطْنِه ) أَي طَرَفَه وآخرَ ما يَبْلغ سِيلانُهُ حِينَ ضرب ، وقيل هو سِيلَانُهُ مُطْلَقاً .<br>\
( و ) صَبِيبٌ : ( ع ) بَلْ هُوَ جَبَل .<br>\
وبه فُسِّر الحَدِيث : ( أَنَّه خَيْرٌ من صَبِيبٍ ذَهَباً ) كما جَاءَ في رواية أُخْرَى مِنْ صَبيرٍ ذَهَباً .<br>\
( أَوْ هُوَ ) صُبَيْبً ( كزُبَيْرٍ ) .<br>\
وقِيلَ : صَبِيبٌ في الحدِيث فَعيلٌ بِمَعْنَى مَفْعُول أَي ذَهَبٌ كَثِير مَصْبُوبٌ غَيْرُ مَعْدُود .<br>\
( والصَّبَابَة : الشَّوْقُ أَو رِقَّتُه ) وحَرَارَتُه ( أَو رِقَّةُ الْهَوَى .<br>\
صَبِبْتَ ) يا رَجُلُ إِليه بالكَسْرِ صَبَابَةً ( كَقَنِعْتَ ) قَنَاعَةً ( فَأَنْتَ صَبٌّ ) أَي عَاشِقٌ مُشْتَاق ( وَهِي صَبَّة ) ومُقْتَضَى قَاعِدَته أَن يَقُولَ وَهِيَ بِهَاءٍ كا تَقَدَّم غَيْر مَرَّة .<br>\
وهذَا الَّذي ذَكَره المُؤَلِّف هو لَفْظ سِيبَوَيْه كما نَقَلَ عَنْه ابْنُ سِيدَه في المُحْكَم والجَوْهَرِيّ في الصَّحَاح ولَا إِجْحَافَ في عبَارَة المؤَلِّف أَصْلاً كما زَعَمِ شَيْخُنَا فنْظُر بالتَّأَمّل .<br>\
وفي لسان العرب : وحَكَى اللِّحْيَانِيّ فِيمَا يَقُولُه نِسَاءُ الأَعْرَابِ عِنْدَ التَّأْخِيذِ بالأُخَذِ : صَبٌّ فاصْبَبْ إِلَيْه ، أَرِقٌ فَارْقَ إِلَيْه .<br>\
قال الكُمَيْتُ :<br>\
ولَسْتَ تَصَبُّ إِلَى الظَّاعِنِينَ<br>\
إِذَا ما صَدِيقُك لم يَصْبَبِ<br>\
وعن ابْنِ الأَعْرَابِيّ : صَبَّ الرجلُ إِذا عَشِق يَصَبُّ صَبَابَةٌ ، ورَجُلٌ صَبٌّ ، ورَجُلَان صَبَّان ، ورجال صَبُّون .<br>\
وامرَأَتَان صَبَّتَان ، ونِسَاءٌ صَبَّاتٌ على مَذْهَب مَنْ قَالَ : رَجُلٌ صَبٌّ بِمَنْزِلَة قَوْلك : رجل فَهِمٌ وحذِرٌ وأَصله صَبِبٌ فاستَثْقَلوا الجَمْعَ بَيْنَ بَاءَيْن متَحَرِّكَتَيْن فأَسْقَطُوا حَرَكَة البَاء الأُولَى وأَدْغَمُوهَا<br>\
في الثانية .<br>\
( و ) الصُّبَيْبُ ( كَزُبَيْر : فَرَسٌ ) من خَيْل العَرَب مَعْرُوفٌ عَنِ ابْنِ دُرَيْد .<br>\
( و ) صَبَّابٌ ( كخَبَّابٍ : جعفْرٌ لبَنِي كِلَابٍ ) نَقَله الصَّاغَانِيُّ وزَادَ غَيْره : كَثِيرُ النَّخْلِ .<br>\
( وصَبْصَبَهُ : فَرَّقَه ومَحَقَه ( وأَذْهَبَه ( فَتَصَبْصَبَ ) وصَبْصَبَ الشيءُ : امَّحَقَ وذَهَبَ .<br>\
( و ) عن أبي عَمْرو : صَبْصَب ( الرَّجُلُ ) إِذَا ( فَرَّقف جَيْشاً أَو مَالاً .<br>\
وصُبَّ ) الرَّجُلُ والشيءُ مَبْنِيًّا لِلْمَجْهُول إِذا ( مُحِقَ ) وهَذَا عَنِ ابْنِ الأَعْرَابِيّ .<br>\
( والتَّصَبصُبُ : ذَهَابُ أَكْثَرِ اللَّيْل ) .<br>\
يقال : تَصَبْصَبَ الليلُ وكذا النهارُ تَصَبْصُباً : ذَهَبَ إلا قَلِيلاً .<br>\
وأَنْشَدَ :<br>\
حَتَّى إِذَا مَا يَوْمُهَا تَصَبْصَبَا<br>\
وعن أَبي عمرو : المُتَصَبْصِبُ : الذاهِبُ المُمَّحِق .<br>\
( و ) التَّصَبْصُبُ : ( شِدَّةُ الجُرْأَةِ والخِلَافِ ) .<br>\
( سقط : يقال تصبصب علينا فلان .<br>\
( و ) التصبصب : ( اشتداد الحر ) .<br>\
قال العجاج :<br>\
حتى إذا ما يومها تصبصبا من صادر أو وارد أيدي سبا قال أبو زيد أي ذهب إلا قليلا ، وقيل أي اشتد على الجمر ذلك اليوم .<br>\
قال الأزهر : وقول أبي زيد أحب إلي .<br>\
ويقال : تصبصب أي مضى وذهب .<br>\
وتصبصب القوم إذا تفرقوا .<br>\
وقال القراء : تصبصب ما في سقائك أي قل .<br>\
( والصبصاب ) بالفتح : ( الغليظ الشديد ، كالصبصب ) كجعفر .<br>\
( والصباصب ) كعلابط .<br>\
ويقال : بعير صبصب وصباصب .<br>\
قال :<br>\
أعيس مصبور القرا صباصب<br>\
( و ) الصبصاب : ( ما بقي من الشيء ) )<br>\
وقال المرَّار :<br>\
تَظَلُّ نِسَاءُ بَنِي عَامِرٍ<br>\
تَتَبَّعُ صَبْصَابَه كُلَّ عَام<br>\
( أَو ما صُبَّ مِنْه ) ، الضَّمِير رَاجِعٌ للشَّيْءِ والمُرَادُ بِهِ السِّقاءُ كَمَا هُوَ في المِحكَمِ وغَيْرِه .<br>\
( و ) قَرَبٌ صَبْصَابٌ : شَدِيدٌ و ( خِمسٌ ) بالكَسْرِ ( صَبْصَابٌ ) مِثْل ( بَصْبَاص ) .<br>\
وعن الأَصْمَعِيّ : خِمْسٌ صَبْصَابٌ وبَصْبَاصٌ حَصْحَاص كُلُّ هَذَا : السَّيْرُ الذي ليست فيه وَتِيرةٌ ولا فُتُور .<br>\
وقد أَحَالَ المُؤَلِّفُ عَلَى الصَّادِ المُهْمَلَة ولا قُصُورَ في كَلَامه كَمَا تَرى كَمَا زَعَمه شَيْخُنا .<br>\
ومِمَّا بَقِي عَلَى المُؤَلِّف مِنْ ضَرُورِيَّات المَادَّة .<br>\
قَوْلُهم مِنَ المَجَازِ : صُبَّ رِجْلَا فُلَانٍ في القَيْدِ ، إِذَا قُيِّدَ .<br>\
قال الفَرَزْدَقُ :<br>\
ومَا صَبَّ رِجْلِي في حديدِ مُجَاشع<br>\
مع القَدْرِ إِلَّا حَاجَةٌ لي أُرِيدُها<br>\
ذكَرَه ابْنُ مَنْظُورِ والزَّمَخْشَرِيّ .<br>\
ومِنَ المَجَازِ أَيْضاً : صَبَّ ذُؤَالَهُ عَلَى غَنَم فُلَانٍ ، إِذَا غَاثَ فيها .<br>\
وصَبَّ اللّهُ عَلَيْهم سَوْطَ عَذَاب إِذَا عَذّبهم .<br>\
وكَذَا صَبّ اللّهُ عَلَيْه صَاعِقَةً .<br>\
ومِنَ المَجَازِ أَيضاً : ضَرَبَه مائَةً فصَبَّا ، مُنَوَّن ، أَيْ فَدُون ذَلِكَ ومائَةً فَصَاعِداً أَي مَا فَوْقَ ذَلِك .<br>\
وقِيلَ صَبًّا مثْل صَاعِداً .<br>\
يُقَالُ : صُبَّ عَلَيْه البَلَاءُ مِنْ صَبَ أَي مِنْ فَوْق ، كَذَا في الأَسَاسِ .<br>\
في لسان العرب عن ابن الأَعْرَابِيّ : ضَرَبه ضَرْباً صَبًّا وحَدْراً ، إِذا ضَرَبَه بحَدِّ السَّيْفِ .<br>\
ومن المجاز أَيضاً : انْصَبَّت الحية على الملدوغ ، إِذا ارتفعت فانْصَبَّت عَلَيْه من فَوْق .<br>\
وهو يصَبُّ إِلى الخَيْر .<br>\
وصَبَّ دِرْعَه : لَبِسَها .<br>\
وانْصَبَّ البازِي عَلَى الصَّيْد .<br>\
تَحَسَّوْا صُبَابَاتِ الْكَرَى .<br>\
كُلُّ ذَلِكَ في الأَسَاس ، وبَعْضُه في لِسَانِ العَرَب .<br>\
وفي التَّهْذِيبِ في حَدِيثِ الصَّلَاة : ( لم يَصُبَّ رَأْسَه ) أَي يُمِلْهُ إِلَى أَسْفَل .<br>\
وفي حَدِيث أُسَامَة : ( فجعل يَرْفَعُ يعدَه إِلَى السَّمَاءِ ثم يَصُبُّها عَلَيَّ ، أَعْرِف أَنَّه يَدْعُو لِي ) .<br>\
وفِي لِسَانِ العَرَب عَنْ أَبِي عُبَيْدَةَ وقَد يَكُونُ الصَّبُّ جمع صَبُوبٍ أَوْ صَابِّ .<br>\
قال الأَزْهَرِيّ ، وقَالَ غَيْرُه : لا يَكُون صَبٌّ جَمْعاً لصَابّ أَو صَبُوب إِنَّمَا جَمْعُ صَابَ أَوْ صَبُوبٍ صُبُبٌ ، كَمَا يُقَالُ : شَاةٌ عَزُوزٌ وعزُزٌ وَجَدُودٌ وجُدُدٌ .<br>\
وفِيهِ أَيْضاً في حَدِيثِ بَرِيرَةَ ( إِنْ أَحَبَّ أَهْلُك أَنْ أَصُبَّ لَهُم ثَمَنَك صَبَّةً وَاحِدَةً أَي دَفْعَةً وَاحِدَةً من صَب المَاءَ يَصُبُّه صَبًّا إِذا أَفْرَغَه .<br>\
وَمِنْهُ صِفَةُ عَليَ لأَبِي بَكْرٍ رَضِيَ الله عَنْهُمَا حِين مات : ( كُنْتَ عَلَى الكَافِرِينَ عَذَاباً صَبًّا ) .<br>\
هو مَصْدَر بمَعْنَى الفَاعِل أَوِ المَفْعُول .<br>\
وماءٌ صَبٌّ كَقَوْلك : مَاء سَكْبٌ ، ومَاءٌ غَوْرٌ .<br>\
قال دُكَيْنُ بْنُ رَجَاء :<br>\
تنْضَحُ ذِفْرَاهُ بمَاءٍ صَبِّ<br>\
مِثْلِ الكُحَيْلِ أَو عَقِيدِ الرُّبِّ<br>\
الكُحَيْلُ : هُوَ النِّفْطُ الَّذِي يُطْلَى به الإِبلُ الجَرْبَى .<br>\
وَفِيهِ في الحَدِيث أَنَّهُ ذكَر فِتَناً فقَالَ : ( لَتَعودُنَّ فِيهَا أَسَاوِدَ صُبًّا .<br>\
يَضْرِبُ بَعْضُكُم رِقَابَ بَعْض ) .<br>\
والأَسَودُ : الحَيَّاتُ .<br>\
وقَوْلُه : صُبًّا .<br>\
قالَ الزّهْرِيُّ وَهُوَ رَاوِي الحَدِيث هُوَ من الصَّبِّ ، قَالَ : والحَيَّةُ إِذَا أَرَادَ النَّهْس ارتَفَع ثُم صَبَّ عَلَى المَلْدُوغِ ، ويُرْوَى صُبَّى بِوَزْنِ حُبْلَى .<br>\
قال الأَزهْرِيّ : قَوْلُه أَسَاوِدَ صُبًّا جَمْعُ صَبُوبٍ وصَبِبٍ ، فحذَفُوا حَرَكَةَ البَاءِ الأُولَى وأَدْغَمُوهَا في البَاءِ الثَّانِيَة فقِيلَ صَبٌّ كما قَالُو الرجل صَبٌّ والأَصْلُ صَبِبٌ ، فأَسْقَطُوا حَرَكَةَ البَاءِ وأَدْغَمُوهَا فقِيلَ صَبٌّ ( كما ) قَالَ .<br>\
قَالَهُ ابنُ الأَنْبَارِيّ ، قَالَ : وَهَذَا هُوَ القَوْلُ في تَفْسِيرِ الحَدِيثِ ، وقَدْ قَالَه الزُّهْرِيّ وصَحَّ عَنْ أَبِي عُبَيْد وابْنِ الأَعْرَابِيّ ، وعَلَيْهِ العَمَلُ<br>\
ورُوِيَ عَنْ ثَعْلَبٍ في كِتَابِ الفَاخِر فقالَ : سُئِل أَبُو العَبَّاسِ عَنْ قَوْله : أَسَاوِد صُبًّا فحدّث عَنِ ابْنِ الأَعْرَابِيّ أَنَّه كَانَ يَقُولُ : أَسَاوِدَ يُرِيدُ ( به ) جَمَاعَات ، سَوَاد وأَسْوِدَة وأَسَاوِد .<br>\
وصُبًّا : يَنْصَبُّ بَعْضُكُم عَلَى بَعْضٍ بالقَتْل .<br>\
وقِيلَ : هُوَ مِنْ صَبَا يَصْبُو إِذا مَالَ إِلى الدُّنْيَا كما يُقَال : غَازٍ وغُزًّا .<br>\
أَرَادَ لَتعُودُنَّ فِيهَا أَسَاوِدَ أَيْ جَمَاعَات مُخْتَلِفِين وطَوَائفَ مُتَنَابِذِين صَابِئين إلى الفِتْنَة مَائِلِين إِلى الدُّنْيَا وزُخْرُفِها .<br>\
قالَ : ولا أَدْرِي مَنْ روى عنه .<br>\
وكَانَ ابْنُ الأَعْرَابِيّ يَقُولُ : أَصْلُه صَبَأَ على فَعَل بالهَمْز مِثْل صَابِىء .<br>\
