وأَعْلِقِ الحَبْلَ بذَيّالٍ زِوَرّ<br>\
( والزِّوَارُ والزِّيَارُ ) ، بالوَاوِ واليَاءِ ( ككِتَاب : كُلُّ شيْءٍ كان صَلاحاً لشَيْءٍ وعِصْمَةً ) ، وهو مَجاز .<br>\
قال ابنُ الرِّقاع :<br>\
كانُوا زِوَاراً لأَهْلِ الشَّامِ قد عَلِمُوا<br>\
لَمَّا رَأَوْا فيهمُ جَوْراً وطُغْيانَا<br>\

( و ) الزِّوَارُ والزِّيَارُ : ( حَبْلٌ يُجْعَل بَيْنَ التَّصْدِيرِ والحَقَبِ ) يُشَدُّ من التَّصْدِير إلى خَلْفِ الكِرْكِرِةِ حتَّى يَثْبُتَ لئلا يُصِيبَ الحَقَبُ الثِّيلَ فيحْتَبس بَوْلُه ، قاله أَبو عَمْرٍ و .<br>\
وقال الفردزق :<br>\
بأَرحُلِنا يَجِدْنَ وقد جَعَلْنَا<br>\
لكلِّ نَجِيبَةٍ منها زِيَارَا<br>\

قال ابنُ الأَثِير : هي جَمْع زِوَارٍ وزِيَارٍ ، المعنَى أَنه جُمِعتْ يَدَاهُ إِلى صَدْرِه وشُدَّت .<br>\
( وزُرْتُ البَعِيرَ ) أَزُورُه زِوَاراً : ( شَدَدْتُه بِه ) ، من ذالك .<br>\
وأَبُو الحُسَيْن ( عَلِيُّ بنُ عَبْدِ الله ابنِ بَهْرَامَ -الزِّيارِيُّ ) الأَسْتَرَابَاذِيّ : ( مُحَدِّثٌ ) يَرْوِي عن إِبراهِيمَ بنِ زُهَير الحُلْوَانيّ ، مات سنة 342 ، كذا في التَّبْصِير للحافِظ ابن حَجَر .<br>\
( والزَّوْرَاءُ ) : اسمُ ( مَال ) كان<br>\
( لأُحَيْحَةَ ) بن الجُلاَحِ الأَنْصَارِيّ : وقال :<br>\
إني أُقِيم على الزَّوْرَاءِ أَعمُرُها<br>\
إِنَّ الكَرِيمَ على الإِخوانِ ذُو المَالِ<br>\
( و ) من المَجَازِ : الزَّوْرَاءُ : ( الْبِئْرُ البَعِيدَةُ ) القَعْرِ .<br>\
قال الشَّاعِر :<br>\
إذْ تَجْعَل الجَارَ في زَوْرَاءَ مُظْلِمَةٍ<br>\
زَلْخَ المُقَامِ وتَطْوِي دُونَه المَرَسا<br>\
وقيل : رَكِيَّةٌ زَوْرَاءُ : غيرُ مُسْتَقِيمَةِ الحَفْرِ .<br>\
( و ) الزَّوْرَاءُ : ( القَدَحُ ) ، قال النابِغَة :<br>\
وتُسْقَى إِذا ما شِئْتَ غَيْرَ مُصَرَّدٍ<br>\
بزوَرَاءَ في حافَاتِهَا المِسْكُ كانِعُ<br>\
( و ) الزَّوْراءُ : ( إِناءٌ ) ، وهو مِشْرَبَةٌ ( من فِضَّة ) مُسْتَطِيلَة مثْل التَّلْتَلَة .<br>\
( و ) من المَجَاز : رَمَى بالزَّوراءِ ، أَي ( القَوْس ) .<br>\
وقَوْسٌ زَوْرَاءُ : مَعْطُوفةٌ .<br>\
( و ) قال الجَوْهَرِيّ : و ( دَجْلَةُ ) بغْدَادَ تُسَمَّى الزَّوْرَاءَ .<br>\
( و ) الزَّوْرَاءُ : ( بَغْدَادُ ) أَو مَدينةٌ أُخْرَى بها في الجانِب الشَّرْقِيّ ؛ ( لأَنَّ أَبوابهَا الدَّاخِلَةَ جُعِلَتْ مُزْوَرّةً ) ، أَي مائلةً ( عن ) الأَبواب ( الخارجَةِ ) وقيل لازْوِرَاء قِبْلَتِها .<br>\
( و ) الزَّوْراءُ : ( ع بالمدينةِ قُرْبَ المَسْجِدِ ) الشريفِ ، وقد جاءَ ذِكرُه في حَدِيث الزُّهْرِيّ عن السَّائِب .<br>\
( و ) الزَّوْرَاءُ : ( دارٌ كانت بالحِيرَةِ ) بناها النُّعمانُ بنُ مُنْذر ، هَدَمَها أَبو جَعْفَر المَنْصُورُ في أَيّامه .<br>\
( و ) الزَّورَاءُ : ( البَعِيدَةُ من الأَراضِي ) قال الأَعْشَى :<br>\
يَسْقِي دِيَاراً لهَا قد أَصْبَحَت غَرَضَاً<br>\
زَوْراءَ أَجْنَفَ عنها القَوْدَ والرَّسَلُ<br>\
( و ) الزَّوْرَاءُ : ( أَرْضٌ عندَ ذي خِيمٍ ) ، وهي أَوَّلُ الدَّهْنَاءِ وآخِرُهَا هُرَيْرَةُ .<br>\
( والزَّارَةُ : الجَمَاعَةُ ) الضَّخْمَةُ ( مِنَ ) الناسِ و ( الإِبلِ ) والغَنَمِ .<br>\
وقيل هي من الأَبل والناسِ : ما بَيْن الخَمْسين إِلى السِّتِّين .<br>\
( و ) الزَّارَة من الطَّائر : ( الحَوْصَلَة ) ، عن أَبِي زَيْد ، ( كالزَّاوِرَةِ ) ، بفَتْح الواو ، ( والزَاوُورَة ) وزاوَرَةُ القَطَاة : ما حَمَلَتْ فيه المَاءَ لِفِراخِها .<br>\
( و ) زَارَةُ : ( حَيٌّ من أَزْدِ السَّرَاةِ ) ، نَقَلَه الصَّاغانِيّ .<br>\
( و ) الزَّارَةُ : ( ة ) كَبِيرَةٌ ( بالبَحْرَيْن ) و ( منها مَرْزُبَانُ الزَّارَةِ ) ، وله حَدِيث معروف .<br>\
قال أَبو مَنْصُور : وعَيْنُ الزَّارَةِ بالبَحْرَيْن مَعْرُوفَةٌ .<br>\
( و ) الزَّارَة : ( ة بالصَّعِيدِ ) ، وسبقَ للمصنّف في ( زرّ ) أَنّها كُورَة بها فلينظر .<br>\
( و ) زَارَةُ : ( ة ، بأَطْرَابُلُسِ الغَرْبِ .<br>\
مِنْهَا إِبراهِيمُ -الزّارِيُّ التاجِرُ المُتَمَوِّلُ ) ، كذا ضَبطه السِّلَفيّ ووَصَفه .<br>\
( و ) زَارَةُ : ( ة من أَعْمَالِ اشْتِيخَنَ .<br>\
منهَا يَحْيَى بن خُزَيْمَةَ -الزَّارِيُّ ) ، ويقال : هي بغَيْر هاءٍ ، رَوَى عن الدَّارميّ .<br>\
وعَنْه طيب بن محمّد السَّمَرْقَنْديّ ، قال الحَافِظُ ابن حَجَر : ضَبَطه أَبو سَعْد الإِدْرِيسيّ هاكذا ، حكاه ابْنُ نُقْطَة .<br>\
وأَمَّا السَّمْعانِيّ فذَكَره بتَكْرِير الزاي .<br>\
( والزِّيرُ ) ، بالكَسْرِ : ( الزِّرُّ ) .<br>\
قال الأَزْهَرِيّ : ومن العَرب مَنْ يَقْلِب أَحَد الحَرْفَيْن المُدْغْمَين ياءً فَيَقُول في مَرَ : مَيْرٌ ، وفي زِرَ زِيرٌ وفي رِزَ رِيزٌ .<br>\
( و ) الزِّيرُ : ( الكَتَّانُ ) .<br>\

وإِنْ غَضِبَتْ خِلْت بالمِشْفَرَين<br>\
سَبَائِخَ قُطْنٍ وزِيراً نُسَالاً<br>\
( والقِطْعَةُ ) منه زِيرَةٌ ، ( بِهاءٍ ) ، والجَمع أَزْوَارٌ .<br>\
( و ) الزِّير : ( الدَّنُّ ) ، والجمع أَزْيَار ، أَعجميٌّ ، ( أَو ) الزِّيرُ : ( الحُبُّ ) الذي يُعْمَل فيه الماءُ ، بلُغَة العِراق .<br>\
وفي حديث الشّافِعِيّ رضي الله عنه ( كُنتُ أَكُتب العِلْم أُلقِيه في زِيرٍ لنا ) .<br>\
( و ) الزِّير : ( العَادَةُ ) ، أَنْشَدَ يُونُس :<br>\
تَقولُ الحارِثِيَّة أُمُّ عَمْرٍ و<br>\
أَهاذا زِيرُه أَبَداً -وزِيرِي<br>\
قال : مَعْناه أَهاذا دَأْبُه أَبداً ودَأْبى .<br>\
( و ) الزِّير : ( رَجُلٌ يُحِبُّ محادَثَةَ النِّساءِ ويُحِبّ مُجالَسَتَهُنَّ ) ومُخَالَطَتَهُنّ ، سُمِّيَ بذالك لكَثْرة زِيَارَتِه لَهُنّ .<br>\
( ويحب ) الثاني مُسْتَدَرَكٌ .<br>\

( بغيرِ شَرَ أَوْ بهِ ) .<br>\

وفي الحديث : ( لا يَزالُ أَحدُكم كاسِراً وِسَادَه يَتَّكِىءُ عليه ويأْخُذُ في الحديث فِعْل الزِّيرِ ) .<br>\

( وهي زِيرٌ أَيضاً ) .<br>\

مَرْيَمُ .<br>\

وزِيرُ المِزْهَرِ مُشْتَقٌّ منه .<br>\
( و ) الزِّيرَةُ ، ( بهاءٍ : هَيْئَةُ الزِّيَارَةِ ) .<br>\

( و ) الزَّيِّر ، ( كسَيِّدٍ ) ، هاكذا في النُّسَخ ، والصَّواب ككَتِفٍ ، كما ضَبَطَه الصَّاغانِيّ : ( الغَضْبَانُ ) المُقَاطِعُ لصاحِبه ، عن ابن الأَعْرَابيّ .<br>\
قال الأَزْهَرِيّ .<br>\
أُرَى أَصْلَه الهَمْزَ ، من زَئِرَ الأَسدُ ، فخفّف .<br>\
( وزُورَةُ ) ، بالضَّمّ ( ويُفْتَحُ : ع قُرْبَ الكُوفَةِ ) .<br>\
( و ) الزَّوْرَة ، ( بالفَتْح : البُعْدُ ) ، وهو من الازْوِرَارِ .<br>\

وماءٍ وَرَدْتُ على زَوْرَةٍ<br>\

ومَاءٍ وَرَدْتُ على زَوْرَةٍ<br>\
