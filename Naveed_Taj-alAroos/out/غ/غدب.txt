<غدب>: ( الغُدْبَةُ بالضَّمِّ ) أَهمَله الجَوْهَرِيّ .
وقال ابنُ دُرَيْد : هي ( لَحْمَةٌ غَلِيظَ ) شَبِيهَةٌ بالغُدَّة تكُونُ ( في لَهَازِم الإِنْسَانِ ) وغَيْرِه .
( و ) قالوا : رَجُلٌ غُدُبٌّ ( كعُتُلَ ) وهو الجَافي ( الغَلِيظُ الكَثِيرُ العَضَل ) ، كحركة .
( وغَدْبَاءُ ) كَصَحْراء : ( ع ) ، قال الشاعر :
ظَلَّت بغَدْبَاءَ بِيَوْمٍ ذِي وَهجْ
( والغُنْدُبَةُ ) بالضَّم يأْتي ذكرها ( في غ ن د ب ) بناءً على أَنَّ النُّونَ أَصْلِيَّة .
