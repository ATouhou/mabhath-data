<عهو>: ( و ( العِهْوُ ، بالكسْر ) : ) أهْملَهُ الجوهرِيُّ .
وقالَ ابنُ سِيدَه : هو ( الجَحْشُ ) ، وكذلكَ العِفْوُ ، والجَمْعُ عهاءٌ .
( و ) العِهْوُ أَيْضاً : ( الجَمَلُ النَّبِيلُ الثَّبَجِ ) ؛ ) وفي بعضِ نسخ المُحْكم : البَلِيلُ الثّبَجِ ، ( اللَّطِيفُهُ ، وهو مَعَ ذلكَ شَديدٌ .
( وأعْهَى ) الرَّجُلُ : ( وَقَعَتْ في مالِهِ ) ؛ ) وفي المُحْكم : في زَرْعِهِ ؛ ( العاهَةُ ) ؛ ) وكذلكَ أَعاهُ وأَعُوه وعَاهُ وعَوَّهُ ؛ عن ابنِ الأعْرابي ؛ كذا في التكْمِلَةِ .
