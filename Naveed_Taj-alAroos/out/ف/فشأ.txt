<فشأ>: ( تَفَشَأَ فيهم المرض ) إِذا ( انتشر ) بهم وعَمَّهم ) ( كَتَفَشَّأَ ) بالشين المعجمة .
قاله أَبو زيد وأَنشد :
وَايمْرٌ عَظِيمُ الشَّأْنِ يُرْهَبُ هَوْلُهُ
وَيَعْيَا بِهِ مَنْ كَانَ يُحْسَبُ رَاقِيَا
تَفَشَّأَ إِخْوَانَ الثِّقاتِ فَعَمَّهُمْ
فَأَسْكَتُّ عني المُعْوِلاَتِ البَواكِيَا
( والفَشْءُ : الفَخْرُ ) قاله ابن بُزُرْجَ ، يقال ( فَشَأَ ) الرجلُ ( كمَنَع  وأَفْشَأَ ) إِذا ( اسْتَكْبَر ) قال أَبو حزام العُكلي : وندُّكَ  مُفْشىءٌ رَيَّخْتُ مِنْهُ :
وَنِدُّكَ مُفْشِيءٌ رَيَّخْتُ مِنْهُ
نَؤُوراً آضَ رِئْدَ نَؤُورِ عُوطِ
( وَتَفَشَّأَ ) فلانٌ ( به ) إِذا ( سَخرَ منه ) واستهزَأَ به .
