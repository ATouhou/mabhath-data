<فنشخ>: ( الفَنْشَخَة ) ، بالشِّين المعجمة بعد النون : العَجْزُ و ( الإِعْيَاءُ والتَّأَخُّرُ عن الأَمرِ ) .
وقد فَنْشَخَ .
وفَشَخَ .
( و ) الفَنْشَخَة : ( التَّفْحِيجُ بَيْنَ الرِّجْلَيْنِ عند البَولِ ) كالفَرْشَحَة ( و ) الفَنْشَخة : ( أَن يَكْبَرَ الرَّجلُ ويَشِيخَ ) ويَعْيَا من الهَرَم .
( و ) من ذالك ( المُفَنْشِخُ ) ، وهو ( السَّاقِط ) على الأَرضِ من الإِعياءِ ( النَّائم ) الكَسْلان .
( و ) من المَجاز : ( تَفَنْشَخَتِ المَرْأَةُ في ) حالة ( الجِمَاعِ ) إِذا ( بَاَدَتْ بَيْنَ رِجْلَيْهَا ) .
( وفَنْشَخٌ ) ، جعفرٍ ، ( عَلَمٌ ) .
[ ] ومما يستدرك عليه من ( التهذيب ) :
يقال : فَنْشَخه فِنْشَاخاً ، وزَلْزَلُه زِلزَالاً بمعنًى واحدٍ .
