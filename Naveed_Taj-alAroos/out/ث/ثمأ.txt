<ثمأ>: ( ثَمَأَهُمْ كجَعَل : أَطْعَمَهم الدَّسَمَ و ) ثَمَأَ ( رَأْسَه ) بالحجر والعَصا ثَمْاً ( : شَدَخُه فَانْثَمَأَ ) وكذلك الثمَر والشجَر .
( و ) ثَمَأَ ( الخُبْزَ ) ثَمْأً ( : ثَرَدَه ) .
( و ) ثمأَ ( الكَمْأَةَ ) ثَمْأً ( : طَرَحَا في السَّمْنِ ) .
( و ) ثَمَأَ لِحْيَتَه ( بالحِنَّاءِ ) ثَمْأً ( : صَبَغَ ) .
( و ) ثَمَأَ ( ما في بَطْنِه : رَمَاه ) واستفرغه .
وكذلك ثَمأَ أَنْفَه : كَسَره فسَال دَماً .
