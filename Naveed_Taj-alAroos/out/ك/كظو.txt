<كظو>: ( و ( كَظَا لَحْمُه ) يَكْظُو : ( اشْتَدَّ ) .
( وفي الصِّحاح : كَثُرَ واكْتَنَزَ .
وفي كتابِ القالِي : يَكْظُو كَظاً : رَكِبَ بعضُه بعضاً .
( وخَظا ) لَحْمُهُ و ( بَظَا ) و ( كَظَا ) : ) كُلُّه بمعْنًى ، وهو ( اتّباعٌ ) .
( قال القالِي : يُكْتَبُ بالألِفِ .
وقد تقدَّمَ خَظَا بَظَا في موْضِعِهِ ، يقالُ ذلكَ ( للصُّلْبِ المُكْتَنِزِ ) ؛ ) قالَهُ الفرَّاء .
( وأَرْضٌ كاظِيَةٌ : يَابسةٌ ) ، وقد كَظَتْ .
( وتَكَظَّى لَحْمُهُ سِمَناً : ارْتَفَعَ ) ؛ ) كذا في التكمِلَةِ .
