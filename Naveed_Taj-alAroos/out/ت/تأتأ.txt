<تأتأ>: ( التَّأْتأَةُ : حِكَايةُ الصَّوْتِ ) تقول : تَأْتَأْتُ به .
( و ) التّأْتَأَةُ ( تَرَدُّدُ التَّأْتَاءِ في التَّاءِ ) إِذا تكلم .
( و ) التأْتأَة ( دُعَاءُ التَّيْسِ ) المِعْزَى ( للسِّفَادِ ) ، وفي ( العُباب ) : إِلى العَسْبِ ( كالتَّأْتَاءِ ) بحذف الهاء .
( و ) التأْتأَةُ ( هي أَضاً مَشْيُ الطِّفْلِ ) الصغيرِ ، وفي ( العباب ) الصَّبِيّ ، بدل الطفْل .
( و ) التأْتَأَةُ ( التَّبَخْتُرُ في الحَرْب ) شجاعةً .
