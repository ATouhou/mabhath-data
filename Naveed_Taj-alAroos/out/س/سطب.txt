<سطب>: ( المَسَاطِبُ ) أَهْمَلَه الجَوْهَرِيّ ، وقَالَ ابن الأَعْرَابِيّ هِيَ ( سَنَادِينُ ) جَمعُ سِنْدَان ( الحَدَّادِين ) .
( و ) المَسَاطِبُ : ) ، المِيَاه السُّدْم ) .
( و ) قال أَبو زَيد : هي ( الدَّكَاكِين يَقْعُد ) النَّاسُ ( عَلَيْهَا .
جَمع مَسْطَبَة ) بفَتْح الميم ( ويُكْسيَرُ ) قال : وسمعت ذلك من العرب .
( والأُسْطُبَّةُ ) بالضم : ( مُشَاقَةُ الكَتَّان ) ، وقد تَقَدَّمَتِ الإِشَارَةُ إِلَيْه في حَرْفِ الهَمْزَة والصَّادِ في كُلِّها لُغَةٌ .
