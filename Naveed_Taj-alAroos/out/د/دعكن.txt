<دعكن>) ( الدَّعْكَنُ ، كجعْفرٍ ) :
أَهْمَلَهُ الجَوْهرِيُّ .
وفي النَّوادِرِ : هو
( الدَّمِثُ الحَسَنُ الخُلُقِ ) مِن الرِّجالِ ؛ نَقَلَهُ الأزْهرِيُّ .
قالَ : ( و ) الدَّعْكَنُ : ( البِرْذَوْنُ ) القَرودُ الأَلْيَسُ البَيِّنُ اللَّيَسِ ( الذَّلُولُ .
( و ) في المُحْكَمِ : الدِّعْكِنَةُ ، ( بهاءٍ : السَّمينَةُ ) .
وقيلَ : ( الصُّلْبَةُ ) الشَّديدَةُ ( مِن النُّوقِ ) ؛ وأَنْشَدَ :
أَلا ارْحَلُوا دِعْكِنةً دِحَنَّهْبما ارْتَعى مُزْهِيةً مُغِنَّهْويُرْوى : ذا عُكْنَةٍ وتقدَّمَ في دَحَنَ ؛ ( ويُكْسَرُ ) وبه رُوِيَ البَيْتُ أَيْضاً .
( و ) الدِّعْكَنَّةُ ، ( كإرْدَبَّةٍ : الحِرُ الضَّخْمُ ) العَظِيمُ .
