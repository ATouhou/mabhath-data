<جلي>: ( ي ( الجِلْيُ ، كعِذْيٍ :
( أَهْمَلَهُ الجوْهرِيُّ .
وقالَ الصَّاغانيُّ : هو ( الكَوَّةُ من السَّطْحِ لا غيرُ .
( وجَلَيْتُ الفِضَّةَ ) جلياً : لُغَةٌ في ( جَلَوْتَها ) ، فهي مَجْلِيةٌ ( واللَّهُ ) تعالى ( يُجَلِّي السَّاعَةَ ) : ) أَي ( يُظْهِرُها ) ، قالَ سبحانه : ( لا يُجَلِّيها لوقْتِها إلاَّ هو ) .
( وتَجَلَّى ) فلانٌ مكانَ ( كذا ) : ) إذا ( عَلاَهُ ) ، والأصْلُ تَجَلَّلَهُ ، قالَ ذو الرُّمَّة :
فلمَّا تَجَلَّى قَرْعُها القاعَ سَمْعَه
وبانَ له وسْطَ الأَشاءِ انْغِلالُها ( و ) تَجَلَّى ( الشَّيءَ : نَظَرَ إليه ) مُشْرفاً ؛ وهذا قد تقدَّمَ في جلو قَرِيباً .
( والمُجَلِّي : السَّابِقُ في الحَلْبَةِ ) والمُصَلِّي الذي يأْتي وَراءَه .
وممَّا يُسْتدركُ عليه :
تَجَلاّهُ الشيء : غَطَّاهُ أَو ذَهَبَ بصبْرِه .
والمجلى : اسمٌ .
وجُلَيَّةُ ، كسُمَيَّةَ : مَوْضِعٌ قُرْبَ وادِي القرَى مِن وَراء شَغْب ؛ قالَهُ نَصْر .
