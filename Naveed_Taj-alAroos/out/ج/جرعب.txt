<جرعب>: ( الجَرْعَبُ ) كجَعْفَرٍ ، أَهمله الجوهريُّ وقال ابنُ دريد : هو ( الجَافِي ، كالجِرْعِيبِ ، بالكَسْرِ .
و ) الجَرْعَبُ ( : الغَلِيظُ ) وفي ( لسان العرب ) هو الجَرْعَبِيبُ ، كخَنْظَلِيل ( و ) الجَرْعَبيبُ ( : الشَّدِيدَةُ مِنَ الدَّوَاهِي ) .
( و ) جَرْعَبٌ ( وَالِدُ جَخْدَبٍ النَّسَّابَةِ ) الكُوفِيّ ، وقد مَرَّ ذِكْرُهُ .
( وجَرْعَبَ المَاءَ : شَرِبَهُ ) شُرْباً جَيِّداً .
( والجُرْعُوبُ ) بالضَّمِّ : الرَّجُلُ الضَّخْمُ الشَّدِيدُ الجَرْعِ لِلْمَاءِ .
( و ) قال الأَزِريُّ : اجْرَعَنّ وارْجَعَنَّ و ( اجْرَعَبَّ ) واجْلَعَبَّ إِذا ( صُرِعَ ) وامْتَدَّ علَى وَجْهِ الأَرْضِ .
