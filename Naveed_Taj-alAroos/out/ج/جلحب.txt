<جلحب>: ( الجِلْحَابِ بالكسر ، و ) الجِلْحَابَةُ ( بهاءٍ ) هو ( الشَّيْخُ الكَبِيرُ ) المُوَلِّي
الهَرِمُ ، وقيل : هُوَ القَدِيمُ ( الضَّخْمُ الأَجْلَحُ ، كالجَلْحَبِ ) مثل جَعفرٍ ( والجُلاحِبِ ) بالضمّ ، نقله ابن السكّبت ( و ) جُلْحَبٌّ ( كقِرْشَبَ ) هو الرجلُ ( الطويلُ ) القَامَةِ ، قالَه أَبو عمرٍ و ، والجِلْحَبُّ أَيضاً : القَوِيُّ الشَّدِيدُ ، قال :
وَهْيَ تُرِيدُ العَزَبَ الجِلْحَبَّا
يَسْكُبُ مَاءَ الظَّهْرِ فيهَا سَكْبَا
والمُجْلَحِبُّ : المُمْتَدُّ ، قال ابن سِيده : وَلاَ أَحُقُّه ، وفي ( التهذيب ) : الجِلْحَابُ : فُحَّالُ النَّخْلِ .
( و ) يقال ( إِبِلٌ مُجْلَحِبَّةٌ ) أَي ( مُجْتَمِعَةٌ ) نقله الصاغانيُّ .
( وجَلْحَبٌ ) كجَعفرٍ ( اسمٌ ) من أَسمائهم .
