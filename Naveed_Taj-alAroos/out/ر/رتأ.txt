<رتأ>: ( رتَأَ العُقْدَةَ ) بالهمز ( كمَنَع ) يَرْتَؤُها رَتْأً و ( رُتُوءًا ) كقُعود ، إِذا ( شَدَّها ) ، كَرَتَاهَا من غير همز ، عن ابنِ دُرَيد .
( و ) رَتَأَ ( فُلاناً : خَنَقَه ) .
( و ) رَتَأَ زيدٌ : ( أَقامَ ) .
( و ) قال الفراءُ : خَرَج يَرْتَأُ شَدِيداً أَي ( انْطلَقَ ) .
( والرَّتَآنُ ) محركةً ممدودةً مثل ( الرَّتَكَانُ ) وزْناً ومعنى .
( وأَرْتَأَ ) الرجلُ : ( ضَحِك في فُتورٍ ) .
( و ) قال ابن شُمَيْل : ( ما رَتَأَ كَبِدَهُ ) اليومَ بِطعامِ أَي ( ما أَكلَ شيئاً ) يَهْجَأَ أَي ( يُسَكِّنُ ) به ( جُوعَه ) قال : وهو ( خَاصٌّ بالكَبِدِ ) أَي لا يقال رَتَأَ إِلاَّ في الكَبدِ ، وكبده منصوب على
المفعولية .
