<حثرب>: ( حَثْرَبَ المَاءُ ) ، أَهْمَلَه الجوهريّ .
وقال ابن السكّيت : أَي ( كَدُرَ و ) كَذَا حَثْرَبَتِ ( البِئرُ ) والقَلِيبُ إِذا ( كَدُرَ مَاؤُهَا واخْتَلَطَ بالحَمْأَةِ ) وفي التكملة : اخْتَلَطَتْ بهِ الحَمْأَةُ ، وأَنشد :
لَمْ تَرْوَ حَتَّى حَثْرَبَتْ قَلِيبُهَا
نَزْحاً وخَافَ ظَمَأً شَرِيبُهَا
( والحِثْرِبَةُ بالكَسْرِ ) لُغَةٌ في ( الحِثْرِمَةِ ) ، قال ابن دُريد : المِيمُ بَدَلٌ عنِ الباءِ ، وهي النَّاتِئةُ في وَسَطِ الشَّفَةِ العُلْيَا منَ الإِنْسَانِ .
( و ) الحُثْرُبُ ( كبُرْقُعٍ ) مثلُ الحُرْبُث ( : نَبَاتٌ سُهْلِيٌّ أَو ) الذي ( لاَ يَنْبُتُ إِلاّ في جَلَدٍ ) من الأَرْضِ ( و ) الحُثْرُب أَيضاً ( : المَاءُ الخَاثِرُ ) ، نقله الصاغانيّ ، ( و : الوَضَرُ ) مُحَرَّكَةً ( يَبْقَى في أَسْفَلِ القِدْرِ ) .
