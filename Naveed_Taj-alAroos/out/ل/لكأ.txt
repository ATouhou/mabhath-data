<لكأ>: ( لَكَأَهُ ) بالسَّوْطِ ( كَمَنَعَه ) لَكْأً ( : ضَرَبَهُ ) ، الليث ، ( و ) في ( التهذيب ) : لَكَأَه كَلَفَأَه ( : أَعْطَاه حَقَّه كُلَّهُ ) عن أَبي عمرو ( و ) لَكَأَه ( : صَرَعَه ) وضَرب به الأَرْضَ .
( و ) لَكِىءَ بالمكان ( كَفَرِحَ : أَقَامَ ) به كَلَكِيَ بغير همز ( و ) لَكِىءَ بالموضع
( لَزِمَ ) ، نقله أَبو عُبَيْدٍ عن الفرَّاءِ ولم يَهْمِزْه غَيْرُه .
( وتَلَكَّأَ عَلَيْهِ ) إِذا ( اعْتَلَّ ، و ) تَلَكَّأَ ( عَنْهُ : أَبْطَأَ ) وَتَوقَّف واعْتَل ، وامْتَنَع ، وفي حديث المُلاعنة : فَتَلَكَّأَتْ عِنْدَ الخَامِسَة .
أَي تَوقَّفَتْ وتَبَاطَأَتْ أَي تَقُولها .
وفي حديث زِيَادٍ : أُتِيَ بِرَجُلٍ فَتَلَكَّأَ في الشَّهادة .
[ ] ومما يستدرك عليه :
قولهم : لَعَن الله أُمًّا لَكَأَتْ بهِ ، أَي رَمَتْ به ، أَي وَلَدَتْه .
