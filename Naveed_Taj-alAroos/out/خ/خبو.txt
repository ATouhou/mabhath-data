<خبو>: ( و ( خَبَتِ النَّارُ ) ؛ وعليه اقْتَصَرَ الجَوهرِيُّ .
زادَ ابنُ سِيدَه :
( و ) كذا ( الحَرْبُ والحِدَّة ) ؛ وفي الأخيرَتَيْن مَجازٌ ؛ يقالُ : خَبَتْ حِدَّةُ النَّاقَةِ تَخْبُو ( خَبْواً ) ؛ بفتْحٍ فسكونٍ ، ( وخُبُوّاً ) ، كعُلُوَ ؛ وعليه اقْتَصَرَ الجَوهرِيُّ ؛ ( سَكَنَتْ .
( و ) في الصِّحاحِ : ( طَفِئَتْ ) .
زادَ ابنُ سِيدَه : وخَمَدَ لَهِيبُها ، وهي خابِيَةٌ .
ومنه قوْلُه تعالى : { كلَّما خَبَتْ زِدْناهُم سَعِيراً } ؛ قيلَ : مَعْناه سَكَنَ لَهِيبُها ؛ وقيلَ : مَعْناهُ كلَّما تَمَنَّوا أن تَخْبُو ، وأَرادُوا : تَخْبُوا .
( وأَخْبيْتُها ) أَنا : ( أَطْفَأْتُها ) وأَخْمَدْتُها ؛ ومنه قوْلُ الكُمَيْت :
ومِنَّا ضِرارٌ وابْنَماهُ وحاجِبٌ
مُؤَجِّجُ نِيرانِ المَكارِمِ لا المُخْبي وممَّا يُسْتدركُ عليه :
خَبَا لَهَبُه : أَي سَكَنَ فَوْرَ غَضَبِه ؛ وهو مَجَازٌ .
