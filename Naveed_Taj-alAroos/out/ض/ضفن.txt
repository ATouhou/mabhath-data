<ضفن>: ( ضَفَنَ إليهم يَضْفِنُ : أَتاهُمْ يَجْلِسُ إليهم ) ؛ ومنه الضَّيْفَنُ الذي يَجِيءُ مع الضَّيْفِ ؛ كذا حَكَاه أَبو عُبَيْدٍ في الأَجْناسِ مع ضفنَ .
وقالَ النحويون : نونُ ضَيْفَن زائِدَةٌ .
( و ) ضَفَنَ ( بغائِطِه ) ضَفْناً : ( رَمَى ) به .
( و ) ضَفَنَ ( بحاجَتِه : قَضَى .
( و ) قالَ أَبو زيْدٍ : ضَفَنَ الرَّجلُ ( المرْأَةَ ) ضَفْناً : ( نَكَحَها .
( و ) ضَفَنَ ( البَعيرُ برِجْلِه خَبَطَ ) بها .
( و ) ضَفَنَ الشَّيءَ ( على ناقَتِه : حَمَلَ ) إيَّاهُ ( عليها .
( و ) ضَفَنَ ( فلاناً : ضَرَبَه بِرِجْلِهِ على عَجُزِهِ ) ؛ وقيلَ : ضَرَبَ اسْتَه بظَهْرِ قَدَمِه ، فهو مَضْفُونٌ وضَفِينٌ .
( و ) ضَفَنَ ( به الأَرضَ ) : إذا ( ضَرَبَها به ) ؛ قالَ الراجزُ :
قَفَنْتُه بالصوتِ أَيَّ قَفْنِوبالعَصا مِن طُولِ سُوءِ الضَّفْنِ ( و ) ضَفَنَ ( ضَرْعَ النَّاقَةِ ) : إذا ( ضَمَّهُ للحَلبِ ) ، عن أَبي زيْدٍ .
( واضْطَفَنَ : ضَرَبَ بقَدَمِهِ مُؤَخَّرَ نَفْسِهِ .
( والضِّفَنُّ ، كهِجَفَ وطِمِرَ : القَصيرُ .
( و ) أَيْضاً : ( الأحْمَقُ في عِظَمِ خَلْقٍ ) ؛ عن الفرَّاءِ ؛ وكذلِكَ ضَفَنْدَدُ ، وكَسْر الفاءِ عنْد ابنِ الأعْرابيِّ أَحْسَن .
( وتَضافَنُوا عليه : تَعاوَنُوا .
( والضَّيْفَنُ : ) مَرَّ ( في الفاءِ ) على أنَّ
النُّونَ زائِدَةٌ ؛ وقد ذُكِرَ هنا ما يُشْتَقُّ منه ، وهو ضفن إليهم .
وممَّا يُسْتدركُ عليه :
الضِّفْنِينُ ، بالكسْرِ : تابعُ الرُّكبانِ ، عن كُراعٍ وحْدِه .
قالَ ابنُ سِيْدَه : ولا أَحُقُّه .
وضَفَنُوا عليه : مالُوا عليه .
وامْرأَةٌ ضِفَنّة ، كهِجَفَّةٍ : حَمْقاءُ رِخْوَة ضَخْمَة ؛ قالَ :
وضِفَنَّةٌ مثلُ الأتانِ ضِبِرَّةٌ ثَجْلاءُ ذاتُ خواصِرٍ ما تَشْبَعُوالضِّفَنَّانُ ، بكسْرٍ ففتحٍ فتَشْديدٍ : الأحْمَقُ الكَثيرُ اللحْمِ الثَّقِيلُ ، والجَمْعُ ضِفْنانٌ ، كقِرْدَانٍ نادِرٌ .
