<بصو>: ( و ( بَصَا ، كدَعَا ) :
وأَهْمَلَهُ الجَوْهرِيُّ .
وقالَ الفرَّاءُ : أَي ( اسْتَقْصَى على غَرِيمِه .
( و ) قالَ أَبو عَمْرو : ( البِصاءُ ، بالكسرْ ) والمدِّ : ( اسْتِقْصَاءُ الخِصَاء .
( و ) قالَ اللَّحْيانيُّ : يُقالُ ( خَصَاهُ اللَّهُ وبَصَاهُ ولَصَاهُ ؛ و ) حَكَى أَيْضاً : ( خَصِيٌّ بَصِيٌّ ) ، ولم يُفَسِّر بَصِيّاً .
قالَ ابنُ سِيدَه : وأُراهُ إِتباعاً ،
( و ) يقالُ : ( ما في الرَّمادِ بَصْوَةٌ ، أَي شَرَرَةٌ ولا جَمْرَةٌ ) .
قُلْتُ : والعامَّةُ تقولُ : بَصْة فيَحْذِفونَ الواوَ .
( وبَصْوَةُ : ع ) ؛ قالَ أَوْسُ بنُ حَجَر :
مِن ماءِ بَصْوَةَ يوماً وهو مَجْهودُ
