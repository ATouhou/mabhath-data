<زمه>: ( الزَّمَهُ ، محرّكةً ) :
( أَهْمَلَهُ الجوْهرِيُّ .
وهي ( لُغَةٌ في الذَّمَهِ ) بالَّذالِ .
يقالُ : ( زَمِهَ الحَرُّ ) ، وذَمِهَ ودَمِهَ ورَمِهَ ( كفَرِحَ ) في الكُلِّ ، إذا ( اشْتَدَّ ) ؛ ) وكَذلِكَ زَمِهَ يومُنا .
( و ) زَمِهَ ( الرَّجلُ بالحَرِّ : اشْتَدَّ عليه ) فآلم دِماغه .
( وزَمَهَتْهُ الشَّمسُ ) ودَمَهَتْه ، ( كمَنَعَ ) : ) آلَمَتْهُ ؛ ( كلُّ ذلك لُغَةٌ في الَّدالِ والَّذالِ ) والَّراءِ .
