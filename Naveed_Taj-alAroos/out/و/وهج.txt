<وهج>: ( وَهَجَ النَّارُ ) ، الصَّواب : وَهَجَت ( تَهِجُ وَهْجاً ) ، بالتسكين ( ووَهَجَاناً ) ، محرَّكةً : إِذا ( اتَّقَدَتْ ) .
ومن المجاز : يَوْمٌ وَهِجٌ ككَتِفٍ ووَهْجانٌ : شَديدُ الحَرِّ .
ولَيْلَةٌ وَهِجَةٌ ووَهْجَانَةٌ ) كذالك .
وقد وَهَجَا وَهْجاً ووَهَجَاناً .
( والاسم الوَهَجُ محرَّكة ) .
( و ) قد ( تَوَهَّجَت ) النارُ : تَوَقَّدَت .
( وأَوْهَجْتُها ) أَنا ( ووَهَّجْتها ) وفي ( المحكم ) : ووَهَجْتُهَا أَنا .
( ولها وَهِيجٌ ) : أَي ( تَوَقُّدٌ ) .
ووَهَجُ الطِّيبِ ووَهِيجُه : انْتِشَارُه وأَرَجُه .
( و ) من المجاز : ( تَوهَّجَتْ رائحةُ الطِّيبِ ) : أَي ( تَوقَّدَتْ ) .
والوَهَجُ والوَهِيجُ : تَلأْلُؤُ الشَّيْءِ وتَوقُّدُه .
( و ) من المَجاز : تَوهَّجَ ( الجَوْهَرُ : تَلأْلأَ ) ، قال أَبو ذُؤيب :
كأَنَّ ابنةَ السَّهْمِيِّ دُرَّةُ قامِسٍ
لها بعدَ تَقْطِيعِ النُّبوحِ وَهِيجُ
والوَهَجُ والوَهْجُ والوَهَجانُ والتَّوَهُّجُ : حَرارَةُ الشَّمسِ والنَّارِ مِن بعيدٍ .
ووَهَجَانُ الجَمْرِ : اضْطِرامُ تَوَهُّجِه .
ونَجْمٌ وَهّاجٌ .
و { سِرَاجاً وَهَّاجاً } ( النبأَ : 13 ) يعني الشَّمْسَ .
والمُتوهِّجَةُ من النِّساءِ : الحَارّةُ المَتاعِ ؛ كذا في ( اللِّسان ) .
